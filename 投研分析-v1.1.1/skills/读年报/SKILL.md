# /读年报 -- A股年报速读与投资备忘录

> 如遇到不熟悉的占位符，参见 [CONNECTORS.md](../../CONNECTORS.md)。

上传A股上市公司年报PDF，自动提取核心财务数据、经营分析、风险提示和重要事项，生成结构化投资备忘录。

**重要提示**：本技能辅助年报阅读和投研工作流程，不构成投资建议。数据来源为用户上传文件，请核实数据准确性。

## Connectors（可选增强）

| Connector | 增强能力 |
|-----------|---------|
| **~~Notion** | 将投资备忘录写入 Notion 知识库 |

> 没有连接器也完全可以使用——直接上传年报PDF即可分析。

## 调用方式

```
/读年报 <年报PDF>
```

分析这份年报：@$1

接受A股年报PDF文件上传。

## 输入要求

接受年报PDF后，确认：

1. **公司名称和代码**：从年报封面提取
2. **报告期**：年度/中报/季报
3. **关注重点**：全面分析/重点看收入/重点看利润/重点看现金流/重点看分红/其他
4. **对比需求**：是否需要与上年同期对比

直接上传年报即可开始，无需额外信息。

## 执行流程

### 第一步：定位核心章节

按照A股年报标准结构，快速定位以下核心章节：

| 章节 | 关注重点 |
|------|---------|
| 第二节 公司简介和主要财务指标 | 核心财务数据速览、股东变化 |
| 第三节 管理层讨论与分析（MD&A） | 收入结构、成本、竞争力、展望 |
| 第四节 公司治理 | 关联交易、对外担保 |
| 第五节 环境和社会责任 | ESG关注点（如适用） |
| 第六节 重要事项 | 诉讼/担保/关联交易/承诺事项 |
| 第十节 财务报告 | 三张报表+附注 |

### 第二步：提取核心财务数据

基于用户上传的年报提取当期数据：
- 如需历史趋势对比，标注"[请提供往年报表]"提示用户补充
- 标注"[请补充股价]"提示用户补充当前股价以计算估值

从财务报表中提取关键指标：

**利润表核心指标**：
- 营业收入及同比增速
- 归母净利润及同比增速
- 扣非归母净利润及同比增速
- 毛利率/净利率/扣非净利率
- 期间费用率（销售/管理/研发/财务费用率）
- 非经常性损益明细（政府补助/投资收益/资产处置/公允价值变动）

**资产负债表核心指标**：
- 总资产/净资产及增速
- 资产负债率
- 应收账款及占收入比（账龄分析）
- 存货及周转天数
- 合同负债（预收款指标）
- 商誉（减值风险）
- 有息负债结构（短期借款/长期借款/应付债券）

**现金流量表核心指标**：
- 经营活动现金流净额及与净利润比较
- 投资活动现金流净额（资本开支方向）
- 筹资活动现金流净额（分红/融资/偿债）
- 自由现金流（经营现金流-资本开支）

**关键财务比率**：
- ROE（加权平均/摊薄）及杜邦分解
- ROIC
- 经营杠杆/财务杠杆
- 营运资本周转效率

### 第三步：分析经营情况

从MD&A章节提取：

**收入结构分析**：按产品/业务线/地区分拆收入，标注各细分的增速和毛利率变化。

**成本端分析**：原材料成本变化、人工成本变化、折旧摊销变化。

**核心竞争力变化**：研发投入及方向、客户集中度、供应商集中度、产能利用率。

**管理层展望**：下一年度经营计划、资本开支计划、行业判断。

### 第四步：行业特化分析

根据所属行业，增加以下差异化关注点：

- **金融行业**（银行/券商/保险）：净息差/手续费收入占比/信用成本/偿付能力充足率/风险加权资产
- **地产行业**：预售数据/土储/竣工面积/融资成本/三道红线指标
- **制造业**：产能利用率/在建工程进度/固定资产周转率/研发费用资本化比例
- **消费行业**：门店数/同店增长/渠道结构/品牌费用投入/库存周转
- **TMT行业**：用户数/ARPU/续费率/研发人员占比/股权激励
- **医药行业**：研发管线进展/集采影响/新品放量/销售费用率

其他行业按一般框架分析。

### 第五步：风险扫描

从年报中提取风险信号，按以下阈值判断严重程度：

| 风险项 | 正常 | 关注 | 警告 |
|--------|------|------|------|
| 审计意见 | 标准无保留 | 带强调事项段 | 保留/无法表示/否定 |
| 应收账款增速-收入增速 | 差值<10pct | 差值10-20pct | 差值>20pct |
| 存货增速-收入增速 | 差值<15pct | 差值15-30pct | 差值>30pct |
| 商誉/净资产 | <10% | 10%-30% | >30% |
| 关联交易/营收 | <5% | 5%-15% | >15% |
| 对外担保/净资产 | <20% | 20%-50% | >50% |
| 股权质押比例 | <20% | 20%-50% | >50% |

额外检查：
- 重大诉讼仲裁、限售股解禁安排
- 会计政策变更（收入确认方式/折旧年限/坏账计提方法变更可能影响利润可比性）
- 审计机构变更（尤其关注连续更换审计机构）
- 新租赁准则（IFRS16/CAS21修订）对资产负债率和EBITDA的影响
- 金融工具分类变更对投资收益和公允价值变动损益的影响

### 第六步：分红分析（如年报含分红方案）

- 每股派息、分红总额、派息率（分红/归母净利润）
- 历史分红趋势（如有MCP数据）
- 股息率计算（如有股价数据）
- 分红方案是否符合公司章程承诺

### 第七步：生成投资备忘录

汇总以上分析，输出结构化投资备忘录。

## 质量标准

- 财务数据须区分"归母净利润"和"扣非归母净利润"
- 非经常性损益必须单独列示（不可混入经营性利润分析）
- ROE须标注计算方式（加权平均/摊薄）
- 应收账款分析须关注账龄结构和坏账计提比例
- 现金流分析须与净利润对比（"现金流质量"）
- 审计意见必须准确识别类型
- 使用申万行业分类
- 不得使用"Non-GAAP"等英文术语，统一用"扣非归母"
- 增速计算须准确（同比=(本期-上期)/|上期|）
- 风险扫描须使用定量阈值判断，不可仅凭主观
