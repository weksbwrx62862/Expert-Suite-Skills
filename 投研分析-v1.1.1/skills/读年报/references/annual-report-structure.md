## 年报标准章节结构

A股上市公司年报按证监会规定格式编制，标准章节如下：

| 章节 | 标题 | 投研关注度 | 关键内容 |
|------|------|----------|---------|
| 第一节 | 重要提示、目录和释义 | 低 | 免责声明、术语定义 |
| 第二节 | 公司简介和主要财务指标 | **高** | 核心财务数据速览、股东变化 |
| 第三节 | 管理层讨论与分析（MD&A） | **高** | 收入结构、成本分析、竞争力、展望 |
| 第四节 | 公司治理 | 中 | 关联交易、对外担保、内控 |
| 第五节 | 环境和社会责任 | 低-中 | ESG相关（部分行业重要） |
| 第六节 | 重要事项 | **高** | 诉讼/担保/关联交易/承诺/分红 |
| 第七节 | 股份变动及股东情况 | 中 | 十大股东、持股变化、质押 |
| 第八节 | 优先股相关情况 | 低 | 如适用 |
| 第九节 | 债券相关情况 | 低-中 | 存量债券信息 |
| 第十节 | 财务报告 | **高** | 三张报表+附注 |

## 快速阅读路径

**15分钟速读路径**（适用于快速筛查）：
1. 第二节：看核心指标变化方向
2. 第三节的MD&A：看管理层自己怎么说
3. 第六节：扫描风险事项
4. 第十节附注：看应收/存货/商誉变化

**60分钟精读路径**（适用于深入分析）：
1. 全部高关注度章节
2. 财务报表逐表分析
3. 附注中的重要会计政策变更
4. 审计报告（关注意见类型）

## 行业差异化关注点

| 行业 | 额外关注章节/附注 |
|------|-----------------|
| 银行 | 贷款五级分类、净息差、拨备覆盖率、资本充足率 |
| 保险 | 保费收入结构、退保率、投资收益率、偿付能力 |
| 券商 | 自营投资收益、经纪业务市占率、投行项目储备 |
| 地产 | 预售数据、土储、竣工/交付、三道红线 |
| 制造 | 产能利用率、在建工程、固定资产变化 |
| 消费 | 门店数/同店增长、渠道结构、品牌费用 |
| 医药 | 研发管线、集采影响、新品销售、学术推广费用 |
| TMT | 用户数/ARPU、续费率/留存、研发人员和费用 |
