# /业绩快评 -- 业绩公告快速点评

上传上市公司业绩公告、业绩快报或业绩预告，快速提取核心数据，与市场预期对比，分析驱动因素，输出点评报告。

**重要提示**：本技能辅助投研工作流程，不构成投资建议。超预期/低预期判断基于可获取的预期数据，请结合最新一致预期验证。

> 如遇到不熟悉的占位符，参见 [CONNECTORS.md](../../CONNECTORS.md)。

## Connectors（可选增强）

> 没有连接器也完全可以使用——财务数据从上传的公告中提取，预期数据由用户手动提供。

## 调用方式

```
/业绩快评 <业绩公告或快报>
```

点评这份业绩公告：@$1

接受业绩公告PDF、业绩快报或直接粘贴业绩数据。

## 输入要求

接受材料后，确认：

1. **公告类型**：年报业绩/中报业绩/季报业绩/业绩快报/业绩预告
2. **公司和代码**：从公告提取
3. **市场预期**：如用户提供Wind/iFinD一致预期则使用，否则标注"无可用一致预期"
4. **关注侧重**：收入端/利润端/现金流/分红/指引

直接上传公告即可开始。

## 执行流程

### 第一步：判断公告类型并提取数据

根据公告类型采用不同提取策略：

**正式财报（年报/中报/季报）**：
- 提取完整财务数据：营收/归母净利/扣非归母/毛利率/净利率/EPS/分红
- 单季度+累计双维度

**业绩快报**：
- 提取初步核算数据（可能与正式报告有差异，标注"[快报数据，正式报告可能调整]"）
- 重点关注收入和利润两个核心指标

**业绩预告**：
- 提取预计净利润区间和变动幅度区间
- 提取管理层说明的变动原因（通常为核心信息）
- 取区间中值与预期对比

核心提取项（适用所有类型）：
- 营业收入及同比增速（单季度+累计）
- 归母净利润及同比增速（单季度+累计）
- 扣非归母净利润及同比增速
- 毛利率/净利率变化
- 每股收益（基本/稀释）
- 分红方案（如有）
- 业绩指引/展望（如有）

### 第二步：预期对比

- 使用用户手动提供的一致预期数据
- 如无一致预期，标注"无可用一致预期，以同比为参照"
- 不包含股价和估值数据；如用户提供股价可折算股息率

**超预期判断标准**：

| 判定 | 条件 | 标注 |
|------|------|------|
| 显著超预期 | 实际值 > 预期值 x 110% | [显著超预期 +X%] |
| 小幅超预期 | 实际值在预期值的105%-110% | [小幅超预期 +X%] |
| 符合预期 | 实际值在预期值的95%-105% | [符合预期] |
| 小幅低预期 | 实际值在预期值的90%-95% | [小幅低于预期 -X%] |
| 显著低预期 | 实际值 < 预期值 x 90% | [显著低于预期 -X%] |

**特殊情况处理**：
- 亏损转盈利：标注"扭亏为盈"，不计算同比增速百分比
- 盈利转亏损：标注"由盈转亏"，计算亏损额
- 上年同期基数极低（净利润<1000万）：标注"低基数效应"，侧重绝对值分析
- 如无一致预期：与上年同期对比，重点分析环比趋势（Q1→Q2→Q3→Q4）

### 第三步：驱动因素分析

分析业绩变动的核心驱动因素：

**收入端**：量价拆分（销量变化 x 价格变化）、新业务贡献、客户结构变化、季节性因素。

**利润端**：毛利率变化（产品结构/原材料/产能利用率）、费用率变化（规模效应/研发投入/一次性费用）、非经常性损益（政府补助/投资收益/资产减值）。

**现金流**：经营现金流与利润匹配度、资本开支变化方向。

**行业差异化关注**：
- **银行**：净息差变化、信用成本、拨备覆盖率、不良率/关注类占比
- **保险**：新业务价值、投资收益率、综合成本率、退保率
- **地产**：结算面积/均价、预售数据、毛利率趋势、净负债率/三道红线
- **消费**：同店增长、门店扩张、渠道库存、会员数据
- **制造**：产能利用率、在手订单、原材料成本、良品率
- **医药**：研发费用率/资本化率、集采影响、新品占比、销售费用率
- **TMT**：用户数/ARPU/LTV、续费率/NRR、研发人员占比

**异常信号检查**：
- 收入增长但应收账款增速远超收入增速 → 可能存在激进确认
- 扣非归母与归母差距过大（非经常性占比>30%） → 盈利质量存疑
- 经营现金流为负但净利润为正 → 利润含金量不足
- Q4单季度利润占全年比例异常（>40%或<0） → 可能存在调节

### 第四步：单季度拆分

将累计数据拆分为单季度数据（如Q1-Q3→拆出单Q3），分析环比趋势。

**拆分公式**：单Qn = 累计Qn - 累计Q(n-1)

判断业绩趋势：
- 环比加速：单季度同比增速逐季提升
- 环比平稳：单季度同比增速基本稳定
- 环比减速：单季度同比增速逐季下降
- 注意：Q4须考虑费用集中确认和资产减值计提的季节性因素

### 第五步：输出点评报告

按中国券商研报体例输出点评。

## 质量标准

- 必须区分"归母净利润"和"扣非归母净利润"
- 单季度拆分计算必须正确（单Q=累计本期-累计上期）
- 同比增速计算须准确，亏损转盈/盈转亏须特殊标注
- 环比趋势判断须有数据支撑
- 无一致预期时须明确标注
- 非经常性损益须单独列示
- 使用中国券商研报体例
- 分红方案须折算股息率（如有股价数据）
- 业绩预告须标注为区间数据，不可当作精确值
