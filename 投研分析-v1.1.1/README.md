# 投研分析

券商/基金投研全流程工具，覆盖深度报告、行业研究、年报解读、业绩快评、调研纪要、晨会纪要和研报摘要七大场景。

> **免责声明：** 本插件辅助专业工作流程，不替代专业意见。所有输出应由具备相应资质的专业人士审核后方可用于决策。

## 能力层级

```
┌──────────────────────────────────────────────────────┐
│  ALWAYS（独立工作）                                    │
│  ✓ 年报/公告/调研笔记等文档的结构化提取和分析           │
│  ✓ 券商体例的深度报告、行业报告撰写框架                 │
│  ✓ 业绩快评、研报摘要、晨会纪要等标准化输出             │
└──────────────────────────────────────────────────────┘
```

## 适用角色

- **行业研究员** — 深度报告和行业研究的核心工具
- **基金经理** — 投资决策的信息处理助手
- **研究所所长** — 晨会纪要和研究管理
- **PE/VC分析师** — 调研纪要和行业跟踪

## Skill 工作流

```
调研纪要 ──→ 深度报告 ──→ 业绩快评（持续跟踪）
  ↑              ↑              ↓
读年报 ────→ 行业研究      研报摘要（对比同行观点）
                               ↓
                           晨会纪要（汇总输出）
```

## 快捷命令

| 命令 | 说明 | 典型输入 |
|------|------|---------|
| `/深度报告` | 公司深度研究报告（首覆/更新） | 公司名称 + 年报/招股书 |
| `/行业研究` | 行业全景研究报告 | 行业名称或细分赛道 |
| `/读年报` | A股年报速读与投资备忘录 | 年报PDF文件 |
| `/业绩快评` | 业绩公告快速点评 | 业绩公告/快报/预告 |
| `/调研纪要` | 调研笔记标准化整理 | 调研笔记或录音转写 |
| `/晨会纪要` | 晨会汇报材料生成 | 覆盖行业 + 素材 |
| `/研报摘要` | 研报核心观点提取与对比 | 1-10份研报PDF |
