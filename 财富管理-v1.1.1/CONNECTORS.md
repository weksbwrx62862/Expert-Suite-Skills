# Connectors

本插件使用类别占位符描述工作流。`.mcp.json` 预配置了具体 MCP 服务，同类别的任何 MCP 服务均可替代。

## 已配置连接器

| Connector | 增强能力 | 受益 Skill |
|-----------|---------|-----------|
| **Notion** | 客户报告、理财规划方案、资产配置方案写入 Notion，方便团队协作和客户交付 | 客户报告、理财规划、资产配置 |

## 无连接器时的工作方式

不安装任何 MCP 连接器，本插件的所有技能均可正常运行：
- 所有分析基于用户提供的数据和文件工作
- 结果以 Markdown 格式输出到对话中
- 连接器仅提供"自动同步到协作工具"的增强体验
