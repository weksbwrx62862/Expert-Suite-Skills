# /市场速览 -- 市场概览速递

> 如遇到不熟悉的占位符，参见 [CONNECTORS.md](../../CONNECTORS.md)。

快速生成结构化市场概览报告，帮助理财经理和客户经理在晨会前或盘后快速掌握市场动态，评估对客户持仓的潜在影响。

**重要提示**：市场数据来源于公开信息，行情数据可能存在延迟，不构成投资建议。

## 能力层级

**独立能力（无需连接器）**

- 通过搜索获取最新市场数据生成速览
- 主要指数+利率+汇率+商品全景
- 资金流向分析（北向/两融/行业主力）+ 信号解读规则
- 行业涨跌归因（政策/业绩/事件/资金/情绪五类驱动）
- 市场估值水位判断（PE百分位分区）
- 对客户持仓的定性/定量影响判断

> 本技能无需连接器即可完整使用——通过搜索获取最新公开市场数据。

## 调用方式

```
/市场速览 [日报|周报] [客户持仓信息]
```

## 输入要求

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 报告模式 | 日报/周报 | 日报 |
| 客户持仓 | 可选，行业/个股/基金 | 通用版 |
| 关注重点 | 权益/固收/商品/全面 | 全面 |

**自动判断规则**：
- 周一至周五盘后 --> 默认日报
- 周末/周一早晨 --> 建议周报
- 用户说"这周""本周" --> 周报

## 执行流程

### 第一步：获取市场数据

1. 通过搜索获取最新行情数据
2. 标注"[公开数据，截至 YYYY-MM-DD]"

### 第二步：指数与宏观速览

**主要指数**：

| 指数 | 观察意义 | 关键技术位 |
|------|---------|-----------|
| 上证综指 | A股整体风向标 | 3000/3200/3500 |
| 深证成指 | 中小盘和成长方向 | -- |
| 创业板指 | 科技成长情绪 | -- |
| 科创50 | 硬科技方向 | -- |
| 恒生指数 | 港股和南向方向 | -- |
| 标普500 | 隔夜美股影响 | -- |
| 纳斯达克 | 全球科技情绪 | -- |

**利率与汇率**：

| 指标 | 观察意义 | 关键阈值 |
|------|---------|---------|
| 10Y国债收益率 | 股债性价比（>3%固收吸引力强） | 2.5%/3.0%/3.5% |
| Shibor 7天 | 短期流动性松紧 | <2%宽松，>3%偏紧 |
| 美元/人民币 | 汇率压力和外资流向 | 7.0/7.2/7.3 |
| 美联储基准利率 | 全球流动性方向 | -- |

**商品**：

| 指标 | 观察意义 |
|------|---------|
| 伦敦金 | 避险情绪+通胀预期 |
| 布伦特原油 | 通胀压力+化工链 |
| 铜 | 全球经济景气度（铜博士） |

**股债性价比信号**（风险溢价 = 1/沪深300PE - 10Y国债收益率）：
- 风险溢价>5% --> 权益吸引力强（历史底部区域）
- 风险溢价3-5% --> 中性
- 风险溢价<3% --> 固收相对更优

### 第三步：市场估值水位

使用最近可得数据。

| 指数 | 当前PE | 近10年百分位 | 估值区间 |
|------|--------|------------|---------|
| 沪深300 | [X] | [X%] | [低估/中性/高估] |
| 中证500 | [X] | [X%] | [低估/中性/高估] |
| 创业板指 | [X] | [X%] | [低估/中性/高估] |

**估值区间判定**：
- 百分位<30% --> 低估（历史相对便宜）
- 30%-70% --> 中性
- >70% --> 高估（注意风险）

**对配置的启示**：
- 全面低估 --> 可适当增加权益仓位
- 全面高估 --> 建议逐步减仓或加强防守
- 分化明显 --> 关注低估方向的结构性机会

### 第四步：资金流向分析

**北向资金**：
- 当日净买入/卖出金额
- 近5日/20日累计趋势
- 重点行业流向

**两融余额**：
- 当前余额+较前日变化
- 趋势判断（增/减/平）

**行业资金流向**：
- 净流入Top5 + 净流出Top5

**资金信号解读规则**：

| 信号 | 含义 | 参考操作 |
|------|------|---------|
| 北向连续3日净流入>50亿 | 外资积极 | 关注其重仓行业 |
| 北向连续3日净流出>50亿 | 外资谨慎 | 警惕短期风险 |
| 两融余额创60日新高 | 杠杆上升 | 波动可能加大 |
| 行业资金高度集中 | 主题行情 | 持续性待观察 |
| 北向与融资同向大幅流入 | 共振做多 | 关注是否过热 |

### 第五步：行业表现与归因

列出涨幅前5和跌幅前5行业，每个附一句话归因。

**归因分析框架**：

| 驱动类型 | 特征 | 持续性判断 |
|---------|------|-----------|
| 政策驱动 | 产业政策/监管变化/补贴 | 看政策力度和落地周期 |
| 业绩驱动 | 财报超预期/景气度上行 | 持续性较强 |
| 事件驱动 | 突发事件/国际局势 | 通常脉冲式 |
| 资金驱动 | 板块轮动/风格切换 | 看资金持续性 |
| 情绪驱动 | 题材炒作/概念发酵 | 通常短期 |

**周报额外内容**：
- 本周行业轮动路径
- 连续3天同方向的持续强/弱势行业
- 下周催化剂日历（财报/政策/数据发布）

### 第六步：市场情绪指标

| 指标 | 当前值 | 含义 |
|------|--------|------|
| 两市成交额 | [X]亿 | <6000亿缩量，>1万亿放量 |
| 涨跌比 | [X:Y] | >3:1强势，<1:3弱势 |
| 涨停/跌停家数 | [X/Y] | 情绪温度计 |
| 换手率(创业板) | [X%] | >3%活跃，<1.5%冷清 |

**综合情绪判断**：
- 量价齐升+涨跌比>2:1 --> 多头主导
- 缩量下跌+涨跌比<1:2 --> 空头主导
- 放量震荡+涨跌比≈1:1 --> 多空分歧

### 第七步：持仓影响分析

**如用户提供了客户持仓**：
1. 逐一匹配持仓行业/个股与当日/本周表现
2. 输出影响矩阵：

| 持仓 | 行业 | 行业表现 | 影响 | 关注点 |
|------|------|---------|------|--------|

3. 综合判断：组合整体受影响方向

**如未提供持仓**：
按客户类型分层建议：

| 客户类型 | 关注要点 |
|---------|---------|
| 保守型(C1-C2) | 利率变化对固收的影响 |
| 平衡型(C3) | 权益固收相对表现，再平衡信号 |
| 进取型(C4-C5) | 行业轮动方向，主题机会 |

### 第八步：输出交付

Markdown格式输出。

## 质量检查清单

- [ ] 主要指数完整（至少A股三大指数+港股+美股）
- [ ] 资金流向有具体金额和趋势判断
- [ ] 行业涨跌有归因分析（非简单罗列）
- [ ] 估值水位有百分位数据和区间判断
- [ ] 情绪指标有量化数据
- [ ] 数据日期和来源已标注
- [ ] 免责声明已包含
- [ ] 一句话摘要<50字
