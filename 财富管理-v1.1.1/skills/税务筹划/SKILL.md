# /税务筹划 -- 合法税务优化方案

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

基于客户收入结构和资产状况，提供合法合规税务优化方案，每个方案附政策依据和精确计算。

**重要提示**：不替代专业税务顾问。严禁任何避税/逃税建议。

## 能力层级

**独立能力（无需连接器）**

- 综合所得税负诊断+边际税率定位
- 年终奖最优策略（含临界点规避）
- 专项附加扣除最大化+夫妻分配优化
- 个人养老金/商业健康险税优
- 投资持有期税务优化

> 本技能无需连接器即可完整使用——基于内置税法知识。

## 调用方式

```
/税务筹划 <收入和资产信息>
```

## 输入要求

1. **收入结构**：工资/劳务/经营所得/财产转让/股息/租金
2. **家庭情况**：配偶收入、子女数/年龄、是否赡养老人
3. **资产状况**：房产/股权/金融资产/保险
4. **特殊情况**：股权激励/境外收入/创业公司股东

## 执行流程

### 第一步：税负诊断

- 列出各项收入+适用税目
- 当前应纳税所得额+边际税率
- 综合税负率 = 应纳个税/税前总收入

### 第二步：优化方案设计

**场景A：工薪族**

策略1：年终奖最优配比
- 单独 vs 合并计税对比
- 最优拆分：令综合所得边际税率 = 年终奖/12的边际税率
- 规避临界点

策略2：专项附加扣除最大化
- 检查所有可享受项目
- 夫妻分配原则：由边际税率更高的一方扣除
- 对比计算确定最优分配

策略3：个人养老金
- 年缴12,000元上限，全额税前扣除
- 节税 = 12,000 x 边际税率
- 边际≥10%时有明显收益(≥1,200元/年)

**场景B：多元收入**

策略1：劳务报酬预扣与汇算退税
- 预扣20-40%，汇算并入综合所得
- 综合税率<预扣率 → 可退税

策略2：经营所得 vs 综合所得
- 个体/独资企业经营所得5-35%
- 综合所得边际>经营所得税率时可优化

**场景C：企业主/股东**

策略1：工资 vs 分红
- 工资3-45%累进 vs 分红20%
- 综合税负：25%企业所得税+20%分红=40%
- 边际>20%时增加分红比例可能更优

策略2：股权激励
- 上市公司：单独计税12月分摊（财税〔2018〕164号）
- 非上市：可递延纳税（财税〔2016〕101号）

**场景D：跨年度调节**
- 预计明年收入降 → 延迟确认
- 预计明年收入升 → 提前确认
- 仅适用可调节收入（劳务/经营所得）

### 第三步：节税效果测算

每个方案：
- 优化前税额(A)
- 优化后税额(B)
- 节税 = A-B
- 验证：各方案节税之和 <= 当前总税负

### 汇算清缴时间线

| 时间 | 操作 |
|------|------|
| 12月 | 确认次年扣除+养老金缴存截止 |
| 1月 | 年终奖计税方式确定（发放时选） |
| 3月1日-6月30日 | 综合所得汇算，多退少补 |
