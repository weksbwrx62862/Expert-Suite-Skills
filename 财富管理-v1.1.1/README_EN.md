# Wealth Management

Full-spectrum wealth management toolkit covering asset allocation, fund analysis, client reporting, financial planning, and tax planning.

> **Disclaimer:** This plugin supports professional workflows and does not replace professional advice. All outputs should be reviewed by qualified professionals before being used for decision-making.

## Capability Tiers

```
+------------------------------------------------------------------+
|  ALWAYS (standalone)                                              |
|  * Asset Allocation: C1-C5 risk profile -> allocation + scenarios |
|  * Fund Analysis: Quarterly PDF / fund code -> holdings + style   |
|  * Client Reporting: Holdings data -> periodic allocation report  |
|  * Financial Planning: Household finances -> education / pension  |
|  * Tax Planning: Income structure -> tax-optimization strategies  |
+------------------------------------------------------------------+
```

## Target Users

- **Relationship Managers** -- Client asset allocation proposals and periodic reports
- **Private Bankers** -- Comprehensive planning and tax optimization for HNW clients
- **Fund Researchers** -- In-depth mutual-fund holdings and performance analysis
- **Family Offices** -- Household wealth succession planning and asset management

## Quick Commands

| Command | Description |
|---------|-------------|
| `/asset-allocation` | Generate a strategic asset allocation plan based on client risk profile and goals |
| `/fund-analysis` | Analyze a mutual fund's holdings, performance attribution, and investment style |
| `/client-reporting` | Generate a periodic allocation report (quarterly / semi-annual / annual) |
| `/financial-planning` | Build a comprehensive household financial plan covering education, retirement, and social insurance |
| `/tax-planning` | Design a lawful personal / household tax-optimization strategy |

## Recommended Workflows

```
/asset-allocation -> /fund-analysis (select specific funds) -> /client-reporting (periodic tracking)
/financial-planning (full client picture) -> /tax-planning (deep-dive on tax efficiency)
```
