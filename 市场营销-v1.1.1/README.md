# 市场营销

市场营销全场景工具，覆盖营销文案、广告合规、竞品追踪、活动策划、社媒热点追踪和SEO优化六大场景。

> **免责声明：** 本插件辅助专业工作流程，不替代专业意见。所有输出应由具备相应资质的专业人士审核后方可用于决策。广告合规检查结果不构成法律建议。

## 能力层级

```
+--------------------------------------------+
|  ALWAYS（独立可用）                          |
|  * 多平台营销文案生成（5大平台适配）            |
|  * 广告法合规扫描（极限词+行业特规）            |
|  * 竞品SWOT分析与差异化建议                    |
|  * 活动方案设计（线上/线下/联动）               |
|  * SEO内容优化（关键词+结构+Meta）             |
|  * 社媒热点筛选与借势建议                      |
+--------------------------------------------+
```

## 适用角色

- **内容运营** — 日常文案撰写、社媒选题（推荐：营销文案 + 社媒热点追踪）
- **品牌经理** — 竞品监控和品牌策略（推荐：竞品追踪 + 广告合规检查）
- **市场总监** — 活动策划和营销全局（推荐：活动策划 + 竞品追踪）
- **增长经理** — SEO和数据驱动增长（推荐：SEO内容优化 + 营销文案）

## 快捷命令

| 命令 | 说明 | 典型输入 |
|------|------|---------|
| `/营销文案` | 多平台适配文案生成 | `/营销文案 新品发布，小红书种草` |
| `/广告合规检查` | 广告法极限词与行业合规扫描 | `/广告合规检查 [粘贴文案]` |
| `/竞品追踪` | 竞品动态分析与定位对比 | `/竞品追踪 瑞幸咖啡` |
| `/活动策划` | 市场活动完整方案设计 | `/活动策划 线下发布会，预算10万` |
| `/社媒热点追踪` | 全网热点采集与借势建议 | `/社媒热点追踪` |
| `/SEO内容优化` | 文章SEO分析与优化 | `/SEO内容优化 [粘贴文章]` |
| `/营销效果分析` | 投放效果ROI分析 | `/营销效果分析 [上传数据]` |
| `/品牌调性审查` | 内容品牌一致性审核 | `/品牌调性审查 [粘贴内容]` |

## Skills

| Skill | 说明 | 核心产出 |
|-------|------|---------|
| 营销文案 | 根据产品信息和目标平台生成适配文案 | 完整文案 + 3个备选标题 + A/B测试建议 |
| 广告合规检查 | 对照《广告法》扫描极限词和违规表述 | 合规报告 + 修改建议 + 合规版文案 |
| 竞品追踪 | 竞品动态分析、定位逆向工程和差异化建议 | 竞品分析报告 + 功能矩阵 + 战斗卡片 |
| 活动策划 | 根据目标和预算设计完整活动方案 | 活动方案 + 预算表 + 传播计划 |
| 社媒热点追踪 | 全网热点采集、借势分析和选题建议 | 热点报告 + 选题建议 + 内容日历 |
| SEO内容优化 | 网站SEO诊断和文章内容优化 | SEO评分报告 + 优化后文章 |
| 营销效果分析 | 营销投放ROI分析和渠道效率评估 | 效果报告 + 漏斗诊断 + 优化清单 |
| 品牌调性审查 | 审查内容与品牌调性的一致性 | 审查报告 + 修改对照 + 渠道适配 |

## Skill 协作路径

```
社媒热点追踪 ──选题──→ 营销文案 ──合规──→ 广告合规检查
                                          ↓
竞品追踪 ──差异点──→ 营销文案         发布前终审
                                          ↓
SEO内容优化 ←──长文──── 营销文案      品牌调性审查
                                          ↓
活动策划 ──复盘──→ 营销效果分析      安全发布
```
