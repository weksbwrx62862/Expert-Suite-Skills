# /营销效果分析 -- 营销投放效果与ROI分析

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

分析营销投放效果，输出核心指标仪表盘、渠道ROI对比、转化漏斗分析、归因模型、趋势预测和优化行动清单，帮助营销团队做出数据驱动的决策。

## 能力概览

### 基础能力（独立可用）
- 核心指标体系搭建（流量 / 转化 / 留存 / 品牌四大维度）
- 渠道ROI对比分析 + 效率象限分类 + 预算再分配建议
- 转化漏斗分析（曝光→点击→访问→留资→转化，逐层诊断瓶颈）
- 归因分析（首次 / 末次 / 线性 / 时间衰减 / U型五种模型对比）
- 趋势分析 + 异常波动识别 + 简易预测
- 中国市场行业基准数据库（电商/教育/金融/游戏/本地生活/SaaS）
- 优化行动清单（按漏斗环节×优化杠杆矩阵排优先级）

## 输入要求

**必填**（以下任一方式）：
1. **上传投放数据 Excel/CSV**：含渠道、投入、曝光、点击、转化等字段
2. **手动输入关键指标**：各渠道的核心数据（投入金额、曝光量、转化数等）

**选填**：
3. **分析时间范围**：本周 / 本月 / 本季度 / 自定义日期范围
4. **对比时间范围**：环比（上周/上月）/ 同比（去年同期）
5. **分析维度**：渠道维度 / Campaign维度 / 时间维度 / 全面分析（默认：全面分析）
6. **业务目标**：获客 / 品牌曝光 / 销售转化 / 用户留存
7. **行业类别**：用于匹配行业基准数据
8. **报告受众**：管理层（精简版）/ 执行层（完整版）

## 执行流程

### 第一步：数据整理与指标体系

#### 1.1 核心指标体系（四大维度）

**流量指标**：
| 指标 | 计算公式 | 健康基准 | 诊断意义 |
|-----|---------|---------|---------|
| UV（独立访客） | — | 取决于渠道 | 流量规模 |
| PV/UV比 | 页面浏览量/访客数 | >2为佳 | 内容粘性 |
| 跳出率 | 单页访问/总访问 | <50%为佳 | 落地页质量 |
| 平均停留时长 | 总停留时间/访问次数 | >2分钟为佳 | 内容吸引力 |

**转化指标**：
| 指标 | 计算公式 | 诊断意义 |
|-----|---------|---------|
| CVR（转化率） | 转化数/访问数 | 整体转化效率 |
| CPA（获客成本） | 总投入/获客数 | 单客获取效率 |
| ROAS（广告回报率） | 广告收入/广告支出 | 广告直接回报 |
| ROI（投资回报率） | (收入-成本)/成本 | 整体投入回报 |

**留存指标**：
| 指标 | 计算公式 | 健康基准 |
|-----|---------|---------|
| 复购率 | 复购用户/总购买用户 | >20%为佳 |
| LTV（客户生命周期价值） | 客单价 x 购买频次 x 生命周期 | LTV>3xCAC为健康 |
| 次日/7日/30日留存 | 回访用户/新用户 | 因产品类型而异 |

**品牌指标**：
| 指标 | 数据来源 | 监测频率 |
|-----|---------|---------|
| 品牌搜索量 | 百度指数 | 周/月 |
| 社媒提及量 | 社媒平台 | 日/周 |
| NPS（净推荐值） | 用户调研 | 季度 |
| 品牌搜索占比 | 品牌词/品类词 | 月度 |

#### 2.2 渠道效率象限分析

按「ROI」和「规模（转化量）」双维度划分：
- **明星渠道**（高ROI + 大规模）：继续加大投入，探索放量天花板
- **潜力渠道**（高ROI + 小规模）：测试放量空间，验证规模化后ROI是否衰减
- **规模渠道**（低ROI + 大规模）：诊断低效原因，优化素材/定向/落地页
- **问题渠道**（低ROI + 小规模）：考虑暂停，释放预算给明星/潜力渠道

### 第四步：归因分析

#### 4.1 归因模型对比

| 归因模型 | 原理 | 适用场景 | 局限性 |
|---------|------|---------|-------|
| **首次触达** | 100%归因首次接触 | 评估获客能力 | 忽略培育和转化环节 |
| **末次触达** | 100%归因最后接触 | 评估临门一脚 | 忽略认知和兴趣环节 |
| **线性归因** | 各触点平均分配 | 公平评估全链路 | 不反映实际贡献差异 |
| **时间衰减** | 越接近转化权重越高 | 平衡性最佳 | 可能低估品牌曝光 |
| **U型归因** | 首末各40%，中间20% | B2B企业推荐 | 权重分配有一定主观性 |

#### 4.2 归因实操建议

- **起步阶段**：先用末次归因（最简单最可执行）
- **进阶阶段**：对比首次和末次，识别「获客型渠道」vs「转化型渠道」
- **成熟阶段**：使用 U型或时间衰减，避免砍掉被低估的辅助渠道
- **注意**：任何归因模型都是近似值，用来指引方向而非精确结论

#### 4.3 归因陷阱提醒
- 品牌曝光类渠道（信息流展示广告）在末次归因中永远表现差——不代表没价值
- 搜索/重定向在首次归因中永远表现差——因为它们本质是收割渠道
- 自报来源（"你从哪里知道我们的"）可做定性参考但不可做定量依据
- 跨设备和跨渠道追踪缺口意味着归因数据永远不完整

## 质量标准

- 所有指标计算必须给出公式，不可只报结论不报过程
- ROI 计算口径需统一说明（是否含人力成本、是否含退款）
- 优化建议必须具体到动作层面（"将SEM出价降低10%"而非"优化SEM"）
- 异常波动需给出至少2个可能原因假设
- 归因分析需说明模型局限性，标注"方向性结论，非精确值"
- 趋势判断需基于至少3个数据点，2个点只能说"暂无趋势"
- 预算再分配建议需标注风险（放量后ROI可能衰减）
- 管理层版本不超过1页/5分钟可消化

## 场景变体

### 周度快报 — 3-5分钟可消化，聚焦核心指标变化+异常+下周重点
### 月度深度分析 — 完整执行所有模块，适合月度复盘会议
### Campaign专项 — 单个活动的完整效果复盘（投前预估 vs 投后实际）
### 渠道评估 — 单一渠道深度分析（如：抖音投放效果专项）
### 预算规划辅助 — 基于历史ROI数据，输出下期预算分配建议和预期产出

## 关联技能

- **活动策划** — 基于效果分析优化下一次活动的渠道选择和预算分配
- **竞品追踪** — 竞品投放策略分析，对标自身投放效率

- **社媒热点追踪** — 社媒渠道效果分析，优化发布时机和内容策略
