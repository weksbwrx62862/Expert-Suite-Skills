# /活动策划 -- 市场活动方案设计

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

根据活动目标和预算，设计完整可落地的市场活动方案，涵盖线上数字营销 campaign、线下执行方案、O2O 联动设计、流程时间线、物料清单、传播计划、预算表、应急预案和复盘框架。

## 能力概览

### 基础能力（独立可用）
- 按目标智能推荐活动类型和预算区间
- 分阶段时间线（筹备 / 预热 / 冲刺 / 活动日 / 发酵 / 复盘）
- 线上数字营销 Campaign 完整设计（渠道矩阵 / 内容日历 / 转化路径 / A/B测试）
- 线下活动执行方案（场地 / 嘉宾 / 流程 / 物料）
- O2O 联动方案设计（线上引流到线下、线下扫码到线上）
- 传播矩阵设计（渠道 x 阶段 x 内容形式 x KPI）
- 物料清单 + 预算编制（含比例参考和应急预留）
- ROI 预估框架
- 应急预案（6 类风险场景 + 应对方案）

### 增强能力（连接 MCP 后解锁）

## 输入要求

**必填**：
1. **活动目标**：品牌曝光 / 获客转化 / 用户激活 / 新品发布 / 行业峰会 / 节日促销 / 周年庆
2. **活动类型**：线上 / 线下 / 线上线下联动

**选填**：
3. **目标受众**：受众画像和预期规模
4. **预算范围**：总预算和弹性空间（未提供则给出三档方案：低/中/高）
5. **时间要求**：活动日期和筹备周期
6. **特殊限制**：场地要求、合规约束、品牌调性

## 执行流程

### 第一步：明确需求与选型

**如果用户未明确活动类型**，按目标推荐：

| 活动目标 | 推荐类型 | 预算基线参考 |
|---------|---------|------------|
| 品牌曝光 | 线上话题营销 + 线下快闪 | 线上 3-10万 / 线下 10-50万 |
| 获客转化 | 线上裂变活动 / 线下体验会 | 线上 1-5万 / 线下 5-20万 |
| 新品发布 | 发布会（线下主 + 线上直播） | 20-100万（含场地/搭建/传播） |
| 行业峰会 | 线下峰会 + 线上直播 | 50-200万（含嘉宾/场地/制作） |
| 节日促销 | 线上限时活动 | 1-10万（主要是推广费） |
| 周年庆 | 品牌事件 + 用户回馈 | 10-50万（含内容制作 + 福利） |
| 私域裂变 | 社群拼团/邀请有礼 | 0.5-5万（主要是奖品成本） |

### 第二步：方案设计

#### 2.1 活动概要
- 活动名称（3 个创意方案供选择，标注各自风格）
- 活动主题和 Slogan
- 核心玩法机制（一句话说清楚用户怎么参与）
- 关键成功指标（KPI）：2-3 个核心指标 + 目标值

#### 2.2 活动流程时间线

**线下活动标准时间线**：
| 阶段 | 时间节点 | 关键动作 | 负责角色 |
|-----|---------|---------|---------|
| 筹备期 | T-30天 | 场地确认、嘉宾邀请、物料设计 | 项目经理 |
| 预热期 | T-14天 | 社媒预热、邀请函发送、媒体联络 | 市场团队 |
| 冲刺期 | T-3天 | 场地搭建、设备调试、彩排 | 执行团队 |
| 活动日 | T-Day | [精确到小时的活动流程] | 全员 |
| 发酵期 | T+1~7天 | 内容发布、媒体报道、社群传播 | 内容团队 |
| 复盘期 | T+14天 | 数据汇总、效果评估、复盘报告 | 项目经理 |

**线上活动标准时间线**：
| 阶段 | 时间节点 | 关键动作 |
|-----|---------|---------|
| 策划期 | T-14天 | 规则设计、页面开发、测试上线 |
| 预热期 | T-3天 | 社媒预告、种子用户激活 |
| 爆发期 | T-Day~T+3 | 活动上线、社群裂变、KOL 推广 |
| 长尾期 | T+3~7 | 二次传播、用户沉淀 |

### 第三步：线上数字营销 Campaign 设计

#### 3.1 渠道策略矩阵

按「付费 + 自有 + 免费」三类渠道规划：

| 渠道类型 | 渠道 | 内容形式 | 目标 | 预算占比建议 |
|---------|------|---------|------|------------|
| **付费渠道** | 信息流广告（巨量引擎/腾讯广告/百度推广） | 短视频/图文/落地页 | 精准获客 | 30-50% |
| **付费渠道** | SEM（百度/360/搜狗） | 搜索广告/品牌专区 | 高意向截流 | 10-20% |
| **付费渠道** | KOL投放（小红书/抖音/B站） | 种草笔记/视频合作 | 信任背书 | 15-25% |
| **自有渠道** | 公众号/视频号 | 推文/短视频/直播 | 私域激活 | 内容成本 |
| **自有渠道** | 小红书企业号 | 种草笔记/互动话题 | 品牌种草 | 内容成本 |
| **自有渠道** | 抖音企业号 | 短视频/直播 | 品牌曝光 | 内容成本 |
| **免费渠道** | SEO内容 | 长文/问答/攻略 | 长尾流量 | 人力成本 |
| **免费渠道** | 社群运营 | 社群活动/UGC | 口碑裂变 | 人力成本 |
| **免费渠道** | 口碑营销 | 用户案例/评价引导 | 信任积累 | 人力成本 |

#### 3.2 内容日历（按活动阶段编排）

| 阶段 | 日期 | 渠道 | 内容主题 | 内容形式 | 目标 |
|-----|------|------|---------|---------|------|
| **预热期** | T-7 | 公众号 | 悬念预告 | 推文+海报 | 制造期待 |
| **预热期** | T-5 | 小红书 | KOL种草铺垫 | 笔记 | 话题预热 |
| **预热期** | T-3 | 抖音 | 倒计时短视频 | 15s视频 | 引流报名 |
| **引爆期** | T-Day | 全渠道 | 活动上线 | 图文+视频 | 集中转化 |
| **引爆期** | T+1 | 信息流广告 | 活动亮点素材 | 短视频广告 | 放大效果 |
| **转化期** | T+2~5 | SEM+落地页 | 限时优惠 | 搜索广告 | 转化收割 |
| **沉淀期** | T+5~7 | 公众号+社群 | 活动回顾+用户证言 | 长文+UGC | 口碑沉淀 |

#### 3.3 转化路径设计

```
流量入口 → 落地页/小程序 → 留资/注册 → 首次转化 → 复购/裂变
   |              |              |            |           |
信息流广告     价值承诺+CTA    低门槛体验   优惠促动    邀请有礼
KOL内容       社会证明        即时反馈     限时紧迫    积分体系
搜索广告       痛点共鸣        信任保障     售后保障    社群运营
```

**各环节转化率基准参考**：
| 环节 | 行业基准 | 优秀水平 | 核心优化点 |
|-----|---------|---------|----------|
| 广告点击率（CTR） | 1-3% | >5% | 素材创意、人群定向 |
| 落地页转化率 | 3-8% | >15% | 首屏价值、CTA设计 |
| 留资→付费 | 5-15% | >25% | 跟进时效、优惠设计 |
| 复购率 | 10-20% | >30% | 服务体验、会员体系 |

#### 3.4 A/B 测试计划

| 测试项 | 变量A | 变量B | 测试指标 | 样本量要求 | 测试周期 |
|-------|------|------|---------|-----------|---------|
| 广告标题 | 利益导向标题 | 痛点导向标题 | CTR | ≥1000曝光/组 | 3天 |
| 落地页首图 | 产品图 | 场景图 | 转化率 | ≥500访问/组 | 5天 |
| CTA文案 | "立即领取" | "免费试用" | 点击率 | ≥300访问/组 | 3天 |
| 优惠策略 | 满减优惠 | 限时折扣 | 客单价 | ≥200订单/组 | 7天 |

> 测试原则：每次只测一个变量，达到统计显著性再下结论。

### 第四步：O2O 联动方案（线上线下联动时使用）

#### 4.1 线上引流到线下

| 引流方式 | 实现路径 | 预期效果 |
|---------|---------|---------|
| 线上报名预约 | 公众号/小程序报名 → 到场核销 | 精准到场率提升 |
| 社群专属邀请码 | 社群发放限量邀请码 → 凭码入场 | FOMO效应驱动 |
| KOL带话题打卡 | KOL预告线下活动 → 粉丝跟随到场 | 借势KOL影响力 |
| 线上任务解锁 | 完成线上互动任务 → 解锁线下权益 | 提升参与粘性 |

#### 4.2 线下引流到线上

| 引流方式 | 实现路径 | 预期效果 |
|---------|---------|---------|
| 现场扫码关注 | 扫码关注公众号/小程序领福利 | 用户留存沉淀 |
| 打卡分享裂变 | 拍照打卡 → 分享朋友圈/小红书 → 领奖品 | UGC内容裂变 |
| 活动专属小程序 | 线下互动环节嵌入小程序操作 | 数据沉淀+后续触达 |
| 直播同步 | 线下活动同步直播 → 线上观众互动 | 扩大活动覆盖面 |

#### 4.3 数据打通设计
- 线上线下统一用户标识（手机号/会员ID）
- 全链路埋点：线上曝光 → 线下到场 → 后续转化
- 归因模型：追踪每个线下到场用户的线上来源渠道

### 第五步：传播计划

按三阶段设计传播矩阵：

| 阶段 | 渠道 | 内容形式 | KPI |
|-----|------|---------|-----|
| 预热 | 微信社群 + 朋友圈 | 悬念海报、倒计时 | 触达 X 人 |
| 预热 | 微博 / 小红书 | 话题引导、KOL 预告 | 话题阅读 X 万 |
| 活动 | 直播平台 | 实时直播、互动抽奖 | 在线峰值 X 人 |
| 活动 | 朋友圈 | 现场打卡素材 | 转发 X 次 |
| 发酵 | 公众号 | 活动回顾长文 | 阅读 X 万 |
| 发酵 | 小红书 / 微博 | UGC 内容、媒体报道 | 二次曝光 X 万 |

#### KOL 选择参考维度
| 维度 | 权重 | 说明 |
|-----|------|------|
| 粉丝量级 | 20% | 匹配活动预算和目标曝光量 |
| 互动率 | 30% | 近30条内容平均互动率，>3% 为优 |
| 调性匹配 | 30% | 内容风格和受众与品牌的契合度 |
| 性价比 | 20% | CPE（单次互动成本）是否合理 |

### 第六步：预算编制与 ROI 预估

#### 预算表
| 类别 | 明细 | 预算 | 占比参考 | 备注 |
|-----|------|------|---------|------|
| 场地 | 租赁 + 搭建 + 设备 | [金额] | 25-35% | 线下活动核心成本 |
| 物料 | 设计 + 印刷 + 周边 | [金额] | 10-15% | 含线上素材设计 |
| 推广 | KOL + 投放 + 媒体 | [金额] | 20-30% | 传播效果关键投入 |
| 嘉宾 | 出场费 + 差旅 + 接待 | [金额] | 10-20% | 峰会/发布会适用 |
| 餐饮 | 茶歇 + 正餐 | [金额] | 5-10% | 按人均 100-300 估算 |
| 应急 | 不可预见费用 | [金额] | **10-15%** | **必须预留** |
| **合计** | | **[总额]** | 100% | |

#### ROI 预估
| 指标 | 计算方式 | 预估值 |
|-----|---------|-------|
| 总投入 | 预算合计 | [X 万] |
| 预计曝光 | 渠道触达 x 预估覆盖率 | [X 万人次] |
| 预计获客 | 曝光 x 预估转化率 | [X 人] |
| 单客获取成本（CAC） | 总投入 / 获客数 | [X 元/人] |
| 预计 ROI | （预计收入 - 投入）/ 投入 | [X%] |

> 如用户未提供转化率基准，按行业均值估算并标注假设条件。

### 第七步：应急预案

| 风险场景 | 概率 | 应对方案 |
|---------|------|---------|
| 天气异常（线下） | 中 | 备选室内场地 / 延期预案 |
| 嘉宾临时取消 | 低 | 备选嘉宾名单 / 调整议程 |
| 设备故障 | 中 | 备用设备 + 技术人员驻场 |
| 报名不足 | 中 | 启动二次推广 + 降低门槛 |
| 现场秩序问题 | 低 | 安保方案 + 志愿者预案 |
| 网络舆情 | 低 | 舆情监控 + 公关话术预案（事实回应，不对抗） |
| 广告投放效果差 | 中 | 备用素材 + 快速调整定向 + 预算转移 |

### 第八步：活动后复盘框架

| 复盘维度 | 核心问题 | 数据指标 |
|---------|---------|---------|
| 目标达成 | KPI 是否完成？差距多少？ | 实际值 vs 目标值 |
| 传播效果 | 哪个渠道 ROI 最高？ | 各渠道 CPE / CPC |
| 用户反馈 | 满意度如何？槽点是什么？ | NPS 评分 / 反馈关键词 |
| 预算执行 | 是否超支？超在哪里？ | 实际支出 vs 预算 |
| 转化链路 | 哪个环节流失最严重？ | 各环节转化率漏斗 |
| A/B测试 | 哪些假设被验证/推翻？ | 测试结果汇总 |
| 经验沉淀 | 下次改进什么？ | Top 3 做对的 + Top 3 要改的 |

## 场景变体

### 线上裂变活动 — 聚焦玩法机制（拼团/砍价/抽奖）、分享链路设计、防薅羊毛规则、裂变系数预估
### 线下发布会 — 聚焦议程设计、媒体邀请、直播搭建、新闻通稿、VIP 接待流程
### 节日营销 — 聚焦节日主题包装、限时机制、情感营销切入点
### 私域裂变 — 聚焦社群分层运营、邀请机制设计、奖励梯度、用户沉淀路径
### 数字营销Campaign — 聚焦渠道组合、素材制作、投放优化、转化路径设计

## 关联技能

- **营销文案** — 活动确定后，为各传播渠道生成配套文案
- **广告合规检查** — 活动宣传物料发布前的合规终审
- **社媒热点追踪** — 活动选题可结合当前热点，借势放大效果
- **竞品追踪** — 了解竞品近期市场动作，设计差异化活动方案
- **营销效果分析** — 活动后数据复盘，各渠道ROI对比分析
- **SEO内容优化** — 活动长尾内容做SEO优化，持续获取搜索流量
