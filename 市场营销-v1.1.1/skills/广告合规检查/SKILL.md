# /广告合规检查 -- 广告法合规扫描

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

对宣传文案进行《广告法》及行业特规合规扫描，识别极限词和违规表述，按风险等级分类，提供合规替代方案和修改后全文。

**重要提示**：本技能辅助内容审核，不替代法务部门的正式合规审批。最终发布决策请咨询专业法律人士。法律法规会更新，请以最新法规文本为准。

## 能力概览

### 基础能力（独立可用）
- 8 类通用违禁词逐句扫描（绝对化 / 虚假承诺 / 极限量词 / 权威误导 / 时限压迫 / 数据无据 / 贬损竞品 / 功效暗示）
- 9 大行业特规深度检查（食品 / 药品 / 医疗器械 / 化妆品 / 金融 / 教育 / 房地产 / 酒类 / 直播带货）
- P0-P3 四级风险评分 + 罚款区间参考
- 渠道差异化规则检查（线上 / 电视 / 社交媒体 / 直播）
- 输出合规版全文（标注所有修改处）

### 增强能力（连接 MCP 后解锁）
- 连接 ~~Notion → 合规检查报告写入 Notion，便于法务审阅签字

## 连接器（可选增强）

## 输入要求

**必填**：
1. **文案内容**：宣传文案全文

**选填**（影响检查深度）：
2. **广告类型**：普通商品 / 食品 / 药品 / 医疗器械 / 化妆品 / 金融 / 教育 / 房地产 / 酒类（默认：普通商品）
3. **投放渠道**：线上 / 线下 / 电视 / 社交媒体 / 直播（默认：线上）
4. **品牌方要求**：是否有内部合规红线或品牌用语规范

## 执行流程

### 第一步：通用违禁词扫描

逐句扫描以下 8 类违规用语：

| 类别 | 典型违规词 | 合规替代 | 法律依据 |
|-----|----------|---------|---------|
| **绝对化用语** | 最佳/第一/唯一/最好/顶级/极致/首选 | 优选/领先/出色/优质 | 《广告法》第9条第3项 |
| **虚假承诺** | 无效退款/100%有效/永不/保证/零风险 | 满意度高/深受好评 | 《广告法》第4条、第28条 |
| **极限量词** | 国家级/世界级/最高级/全网最低/史上最强 | 高品质/广受认可 | 《广告法》第9条第3项 |
| **权威性误导** | 国家机关推荐/指定/特供/专供 | （删除，不可替代） | 《广告法》第9条第2项 |
| **时限压迫** | 仅此一次/错过不再/最后机会 | 限时优惠/本期特惠 | 《广告法》第28条 |
| **数据无据** | 销量第一/行业领先/遥遥领先（无来源） | 补充数据来源或删除 | 《广告法》第11条 |
| **贬损竞品** | 比XX好/XX做不到/碾压XX | （删除，改为正面表述） | 《广告法》第13条 |
| **功效暗示** | 药妆/医学护肤/祛斑美白（化妆品）/包治百病 | 按行业规定替换 | 《广告法》第17条 |

### 第二步：行业特殊规定检查

**根据广告类型执行对应的行业检查**：

#### 食品广告
- [ ] 不得宣称治疗、预防疾病功能（《广告法》第17条）
- [ ] 不得使用"纯天然""零添加"等绝对性表述
- [ ] 保健食品必须标注"本品不能代替药物"
- [ ] 婴幼儿食品不得宣称增智/增高功能

#### 药品/医疗器械广告
- [ ] 必须标注禁忌、不良反应（《广告法》第16条）
- [ ] 不得使用患者/医生形象做推荐
- [ ] 不得承诺治愈率、有效率
- [ ] 处方药禁止面向公众广告

#### 化妆品广告
- [ ] 不得宣称医疗效果（《广告法》第17条）
- [ ] 不得使用"药妆""医学护肤"等暗示医疗的表述
- [ ] 功效宣称需有检测报告支撑
- [ ] 美白/祛斑等特殊功效需备案

#### 金融广告
- [ ] 必须提示投资风险（"投资有风险，选择需谨慎"）
- [ ] 不得承诺保本保收益
- [ ] 不得使用"稳赚""躺赚"等误导性表述
- [ ] 贷款广告必须明示年化利率

#### 教育广告
- [ ] 不得承诺通过率、就业率
- [ ] 不得使用"包过""包就业"等保证性表述
- [ ] 不得利用学员升学/考试成绩做广告

#### 房地产广告
- [ ] 不得含有升值/投资回报承诺
- [ ] 面积标注必须明确计算方式
- [ ] 不得以直线距离表示项目位置
- [ ] 预售项目需标注预售许可证号

#### 酒类广告
- [ ] 不得诱导饮酒或宣传饮酒有益健康
- [ ] 不得出现饮酒驾车等危险行为暗示
- [ ] 不得面向未成年人（含投放渠道审查）

#### 直播带货
- [ ] 主播口述是否构成广告代言（《互联网广告管理办法》）
- [ ] "全网最低价""独家价格"等是否有依据
- [ ] 限时限量话术是否构成虚假信息
- [ ] 主播是否具备特定品类推荐资质（如药品）

### 第三步：渠道差异化规则

| 检查项 | 线上图文 | 社交媒体 | 直播 | 电视 |
|-------|---------|---------|------|------|
| 广告标识 | 页面需标注"广告" | 推广笔记需标注"合作" | 需口头或字幕提示 | 需声明"以下是广告" |
| 用户评价引用 | 需标注来源和时间 | 同左 | 口播引用需说明来源 | 不得使用个人证言 |
| 价格表述 | 需标注有效期 | 同左 | 当场有效即可 | 需标注有效期 |

### 第四步：风险评分

为每处问题评定风险等级：

| 风险等级 | 定义 | 处理建议 | 罚款参考 |
|---------|------|---------|---------|
| **P0-严重** | 明确违反禁止性条款 | 必须修改，不得发布 | 20-100万或营业额3-5倍 |
| **P1-高** | 违反行业特规或虚假宣传 | 强烈建议修改 | 5-20万 |
| **P2-中** | 表述不严谨，有被举报风险 | 建议修改 | 1-5万（视情节） |
| **P3-低** | 用语欠妥但法律风险较小 | 可选修改 | 警告或整改 |

### 第五步：合规改写

改写原则（保效果又合规）：
1. **保留核心卖点**：只替换违规词，不削弱营销力度
2. **用数据替代夸张**：将"销量第一"改为"累计服务10万+用户"（需有数据支撑）
3. **用用户视角替代品牌视角**：将"最好的产品"改为"深受用户好评"
4. **增加限定条件**：将"永不过期"改为"在正常存储条件下保质期长达X年"

## 质量标准

- 扫描必须覆盖《广告法》全部禁止性规定及对应行业特规
- 每处违规必须引用具体法条和款项
- 替代方案必须既合规又保持营销效果
- **宁可多报不可漏报**——边界模糊的表述宁可标记为 P3 也不要忽略
- 合规版文案要完整可用，不能只改违规处而忽略上下文连贯
- 不得遗漏渠道特殊要求（特别是直播和社交媒体的广告标识要求）

## 平台审核机制速查

不同平台除法律合规外，还有各自的广告审核规则：

| 平台 | 审核特点 | 高频被拒原因 | 建议 |
|-----|---------|-----------|------|
| 抖音/巨量引擎 | 机审+人审，素材审核严 | 画面含二维码、诱导关注、绝对化表述 | 避免"最""第一"，画面干净 |
| 小红书 | 对"软广"敏感 | 未标注"合作"、引流外站、虚假种草 | 合作笔记必须报备 |
| 微信广告 | 落地页合规要求高 | 落地页与广告不一致、表单过度收集 | 广告内容≤落地页承诺 |
| 百度推广 | 行业资质审核严 | 缺少行业资质、夸大效果 | 提前准备好资质文件 |
| 快手 | 对直播带货话术敏感 | "全网最低""假一赔十"无证据 | 限时优惠需说明限时条件 |

## 场景变体

### 单篇深度检查 — 默认模式，对一篇文案逐句扫描，输出完整报告和合规版全文
### 批量快速扫描 — 用户给出多条文案（如一组朋友圈文案），每条给出"通过/需改"结论和关键问题
### 直播话术检查 — 聚焦主播口播稿，检查代言合规、价格表述、限时限量话术
### 投放素材审核预检 — 按目标投放平台的审核规则预检，降低被拒概率
