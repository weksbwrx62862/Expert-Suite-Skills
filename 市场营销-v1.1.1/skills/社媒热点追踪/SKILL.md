# /社媒热点追踪 -- 全网热点与借势选题

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

聚合全网热点话题，评估借势安全性和品牌匹配度，输出可执行的内容选题建议和多平台内容日历。

## 能力概览

### 基础能力（独立可用）
- 七大平台热榜聚合（微博 / 微信 / 抖音 / 小红书 / 知乎 / B站 / 头条）
- 三级安全筛选（绝对禁区 / 谨慎评估 / 可借势）
- 五维量化评估模型（相关性 / 时效性 / 安全性 / 可创作性 / 品牌契合度）
- 热点生命周期预判（萌芽→上升→峰值→衰退，预估剩余窗口）
- 平台算法推荐机制适配（各平台热度衰减曲线和推荐权重差异）
- 每个热点输出切入角度 + 标题建议 + 发布时机
- 多平台内容日历（覆盖 3-7 天）

### 增强能力（连接 MCP 后解锁）
- 连接 ~~Notion → 热点报告和选题数据库写入 Notion，团队即时同步

## 输入要求

**选填**（未提供则通用分析）：
1. **行业/品牌**：所属行业和品牌名称（用于匹配相关热点）
2. **目标平台**：重点关注的平台（微博 / 微信 / 抖音 / 小红书 / 知乎 / B站）
3. **内容方向**：品牌调性和内容禁区（如"科技感，不做娱乐八卦"）
4. **关注热点类型**：社会民生 / 娱乐 / 科技互联网 / 财经商业 / 体育赛事 / 全部
5. **紧急程度**：日常监控 / 紧急追热（影响输出详细程度）

## 数据获取方式

本技能的热点数据通过以下方式获取：

### 方式一：搜索获取（默认）

通过搜索获取各平台热搜榜单和热点话题。适合大多数场景。

### 方式二：浏览器插件增强（推荐日常高频使用者）

QoderWork 浏览器插件可让本技能直接读取各平台网页的实时热榜数据，获得更完整、更及时的信息。

**安装步骤**：
1. 打开 QoderWork 桌面端，进入「设置 → 浏览器插件」
2. 点击「安装 Chrome 插件」或「安装 Edge 插件」，跳转至浏览器扩展商店
3. 安装完成后，在浏览器工具栏点击 QoderWork 图标，登录同一账号完成绑定
4. 回到 QoderWork 对话界面，当你调用 /社媒热点追踪 时，agent 会自动通过插件访问各平台网页

**安装后的增强效果**：
| 能力 | 无插件 | 有插件 |
|-----|--------|--------|
| 微博热搜实时榜 | 搜索间接获取（有延迟） | 直接读取网页数据（实时） |
| 抖音/快手热榜 | 搜索获取（可能不完整） | 直接读取完整榜单 |

| 热点详情页 | 仅标题级信息 | 可读取讨论量、参与人数等详情 |

**使用技巧**：
- 在对话中说"帮我看看微博热搜"，agent 会通过插件打开微博热搜页面并读取数据
- 你也可以自己在浏览器中打开热搜页面，然后说"分析我当前看到的热搜"

## 执行流程

### 第一步：热点采集

使用 agent 自带搜索能力，搜索以下关键词聚合热点：
- "今日热搜""微博热搜榜""知乎热榜""抖音热点""B站热门""小红书热搜""36Kr 热榜"

按热度指数排序，去重合并跨平台重复话题，标注各话题来源和热度趋势。

**如果用户主动提供了热搜截图或关键词**：
1. 优先基于用户提供的信息分析
2. 用搜索能力补充验证和获取更多上下文

#### 各平台热榜特征与算法机制

| 平台 | 热度机制 | 热度衰减 | 核心用户 | 借势内容偏好 |
|-----|---------|---------|---------|------------|
| 微博 | 实时搜索量 + 讨论量 | 快（4-8小时） | 18-35岁，话题驱动 | 话题讨论、观点输出、段子 |
| 微信 | 公众号阅读 + 搜一搜指数 | 慢（24-48小时） | 25-45岁，深度内容 | 深度分析、知识科普 |
| 抖音 | 播放量 + 互动率 + 完播率 | 快（6-12小时） | 18-35岁，娱乐驱动 | 短视频跟拍、挑战赛 |
| 小红书 | 收藏率 + 互动率 | 慢（3-7天长尾） | 18-35岁女性为主 | 种草笔记、经验分享 |
| 知乎 | 浏览量 + 回答数 + 赞同率 | 慢（48-72小时） | 25-40岁，知识型 | 专业回答、深度解读 |
| B站 | 弹幕量 + 投币数 + 三连率 | 中（12-24小时） | 15-30岁，Z世代 | 视频解说、二创内容 |
| 头条 | 点击率 + 阅读时长 + 转发 | 快（6-12小时） | 25-50岁，资讯消费 | 快讯解读、观点文章 |

> **算法启示**：微博/抖音/头条热度衰减快，需在2小时内响应；小红书/知乎热度长尾，可在峰值后1-2天发布深度内容仍有流量。

### 第二步：热点生命周期预判

对每个热点判断其所处阶段，决定响应策略：

| 阶段 | 特征信号 | 剩余窗口 | 响应策略 |
|-----|---------|---------|---------|
| **萌芽期** | 单平台上榜、讨论量<1万 | 12-24小时 | 提前布局，做第一批内容（高风险高收益） |
| **上升期** | 多平台上榜、讨论量快速增长 | 6-12小时 | 最佳借势窗口，立即出稿 |
| **峰值期** | 全平台热搜、讨论量触顶 | 2-4小时 | 差异化切入（独特角度），避免同质化 |
| **衰退期** | 热度下降、新话题替代 | <2小时 | 仅深度内容平台（知乎/公众号）可发，短视频平台放弃 |
| **长尾期** | 热搜下榜但搜索量仍在 | 3-7天 | 适合SEO型内容（知乎/公众号），获取搜索流量 |

**预判方法**：
- 上榜时长 > 6小时且热度仍在上升 → 上升期
- 热度指数在2小时内翻倍 → 即将进入峰值
- 已有多个大V/官媒发表观点 → 接近峰值或已过峰值
- 新话题开始抢占热搜位 → 进入衰退期

### 第三步：安全筛选（三级过滤）

#### 绝对禁区（一律排除，不做借势推荐）
- 政治敏感 / 国际关系 / 军事冲突
- 自然灾害 / 重大事故 / 人员伤亡
- 社会负面事件（校园霸凌 / 家暴 / 犯罪）
- 民族 / 宗教 / 性别争议
- 未经证实的传闻 / 阴谋论
- 涉及未成年人的争议事件

#### 谨慎评估（标注风险等级，由用户决定）
- 明星绯闻 / 娱乐争议 → 风险：粉丝反噬
- 行业负面新闻 → 风险：被关联
- 社会观点争议 → 风险：站队翻车
- 品牌翻车事件 → 风险：幸灾乐祸形象

**防错规则**：如无法确认话题完整背景，一律归入「谨慎评估」而非「可借势」。

#### 可借势（通过安全筛选，进入评估）
- 正面社会新闻、节日节气、行业利好、文化热点、体育赛事、影视综艺

### 第四步：借势价值评估

对通过安全筛选的热点，按五个维度打分（每项 1-5 分）：

| 评估维度 | 1分 | 3分 | 5分 | 权重 |
|---------|-----|-----|-----|------|
| **相关性** | 与品牌无关联 | 可牵强关联 | 天然强关联 | x2 |
| **时效性** | 已过峰值 | 峰值期 | 上升期 | x1 |
| **安全性** | 有翻车风险 | 风险可控 | 完全安全 | x2 |
| **可创作性** | 难以产出 | 角度有限 | 角度丰富 | x1 |
| **品牌契合度** | 调性不符 | 勉强匹配 | 高度契合 | x2 |

**综合评分 = 相关性x2 + 时效性 + 安全性x2 + 可创作性 + 品牌契合度x2**（满分 40 分）
- 32-40分：强烈推荐借势
- 22-31分：可以借势
- <22分：不建议借势

### 第五步：输出选题建议

对推荐借势的热点，提供具体内容创意：

每个热点给出：
- **切入角度**：从什么视角关联品牌（正面借势 / 反向解读 / 知识延伸 / 情感共鸣）
- **内容形式**：图文 / 短视频 / 直播 / 互动话题
- **标题建议**：2-3 个适配目标平台的标题
- **发布时机**：最佳发布窗口（精确到小时）
- **预期效果**：基于热度和关联度的流量预估（高/中/低）

#### 黄金发布时间参考

| 平台 | 工作日最佳 | 周末最佳 | 算法加权时段 |
|-----|----------|---------|-----------|
| 微博 | 12:00-13:00, 18:00-20:00 | 10:00-12:00 | 热搜上升期跟发1小时内 |
| 公众号 | 07:00-08:00, 20:00-22:00 | 09:00-10:00 | 发布后2小时阅读量决定推荐 |
| 抖音 | 12:00-14:00, 19:00-22:00 | 全天均可 | 发布后1小时完播率决定流量 |
| 小红书 | 12:00-14:00, 19:00-21:00 | 10:00-12:00 | 发布后2小时互动率决定推荐 |
| 知乎 | 10:00-12:00, 20:00-23:00 | 14:00-17:00 | 热榜问题发布后6小时内回答 |

> **关键认知**：热点借势的时间窗口通常只有2-6小时。超过窗口期，内容质量再高也难获得算法推荐的热度加成。

## 质量标准

- **安全第一**：宁可错过热点，不可踩雷翻车
- 借势角度必须自然，禁止强行关联（如灾难话题关联促销）
- 标注热点所处生命周期阶段，避免追已过时话题
- 风险评估必须保守——边界模糊的一律标注风险
- 内容日历至少覆盖 3 天，标注各平台最佳发布时间
- 评分必须逐项列出各维度得分，不可只给总分
- 紧急追热场景下，安全筛选不可省略

## 场景变体

### 日常监控 — 默认模式，全面扫描当日热点，输出完整报告和内容日历
### 紧急追热 — 用户指定某个正在爆发的热点，快速评估安全性并给出即时内容方案（10分钟出稿级别的快速建议）
### 周度选题规划 — 结合本周可预见的节点（节日/赛事/行业事件），提前规划 7 天内容日历
### 竞品热点监控 — 监控竞品近期借势了哪些热点、效果如何，指导自身策略

## 关联技能

- **营销文案** — 确定选题后，直接调用生成适配平台的完整文案
- **广告合规检查** — 借势内容发布前做合规扫描，避免热点文案踩《广告法》红线
- **SEO内容优化** — 热点类长文可做 SEO 优化，获取搜索长尾流量
- **品牌调性审查** — 借势热点内容发布前，确认是否符合品牌调性
- **竞品追踪** — 竞品借势动态也是重要市场信号
