# 专家SKILL

一站式AI专家套件集合，覆盖金融、法律、财务、营销、咨询、产品、编程等多个专业领域。每个套件将岗位专业知识封装为可一键调用的AI Skill，让大语言模型成为真正的行业专家。

## 项目概览

本项目包含11个专业领域的专家套件，每个套件都包含：
- **Skills（技能）**：岗位核心工作场景的AI指令模板
- **Connectors（连接器）**：与外部工具（Notion等）的集成配置
- **Plugin配置**：QoderWork插件标准格式的元数据定义

```
专家SKILL
├── 专家套件创建指导      # 如何创建新的专家套件
├── 产品管理-v1.1.1       # 产品经理全流程工具
├── 企业法务-v1.1.1       # 企业综合法务助手
```

## 快速开始

### 使用已有套件

1. 进入对应套件目录，查看 README.md 了解该套件的完整能力
2. 每个 Skill 都有独立的 SKILL.md，包含触发条件和详细指令
3. 通过 `/Skill名称` 格式触发对应能力（如 `/PRD生成`、`/代码审查`）

### 创建新套件

参考 [专家套件创建指导](专家套件创建指导/SKILL.md)，按照以下流程创建：

```
岗位分析 → Skill设计 → 资源准备 → 文件编写 → 打包验证
```

## 技术架构

每个专家套件遵循统一的QoderWork插件标准：

```
套件目录/
├── .qoder-plugin/
│   └── plugin.json          # 插件元数据定义
├── .mcp.json                # MCP连接器配置
├── README.md                # 套件说明文档
├── CONNECTORS.md            # 连接器详细说明
└── skills/
    └── 技能名称/
        ├── SKILL.md         # 技能定义（触发条件+指令模板）
        └── references/      # 技能参考材料（可选）
```
