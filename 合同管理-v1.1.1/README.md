# 合同管理

<!-- [优化:场景] 重写插件描述，突出AI增量价值和端到端能力 -->
合同全生命周期管理助手：帮你审查合同风险、起草合同初稿、对比合同版本差异、快速分诊NDA、检索法条依据、管理合同到期提醒。基于《民法典》合同编及相关司法解释，覆盖企业常见合同场景。

> **免责声明：** 本插件辅助专业工作流程，不替代法律意见。所有输出应由执业律师审核后方可用于决策。

## 适用角色

- **企业法务** -- 日常合同审查和管理的核心工具
- **法务总监** -- 快速评估合同风险，制定谈判策略
- **采购经理** -- 供应商合同审查和谈判准备
- **创业者/CEO** -- 没有法务团队时的合同审查第一道防线

## Skills

<!-- [优化:知识] 补充每个 Skill 的具体功能描述 -->

| Skill | 说明 |
|-------|------|
| 审查合同 | 上传合同文件，逐条审查风险并按红黄绿分级标记，输出修改建议和三梯队谈判策略 |
| 起草合同 | 描述交易背景，生成符合中国法律要求的合同初稿 Word 文档，关键条款附法律依据 |
| 合同对比 | 上传两版合同，逐条语义级对比差异并按影响等级分类，含联动分析和谈判建议 |
| NDA快筛 | 上传NDA文件，八项关键条款快速筛查，30秒给出"可签/需改/需谈"分级判断 |
| 法条速查 | 输入业务问题或关键词，检索匹配法条、司法解释和典型案例要点 |
| 合同台账提醒 | 录入合同日期信息，自动生成到期提醒、续签提醒和履约节点追踪 |
