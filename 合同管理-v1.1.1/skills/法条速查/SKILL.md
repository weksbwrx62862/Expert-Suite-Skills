# /法条速查 -- 法律条文智能检索

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

根据业务问题或关键词，智能检索匹配的法律条文、司法解释和典型案例，输出实务解读和操作建议。

**重要提示**：本技能辅助法律研究，不构成法律意见。检索结果应由执业律师审核确认。

**独立能力（无需连接器）**

- 业务问题到法律问题的智能转换
- 核心法条 + 关联司法解释 + 典型案例
- 三层实务解读（条文原文/要点解读/实务要点）
- 多法条竞合时的适用优先级判定
- 高频企业法律问题速查索引

**增强能力（连接器加持）**

- ~~Notion -> 将法条速查结果发布到 Notion，便于团队共享和存档

|---------|------|---------|
| **业务问题** | "员工入职后发现简历造假，能否解除合同？" | 提取法律要素，检索相关法条 |

## 执行流程

### 第一步：解析检索意图

分析用户输入，识别：
- **核心法律问题**：用户真正想了解什么
- **法律部门**：民商法/劳动法/行政法/知识产权法/刑法等
- **关键法律概念**：涉及的核心法律术语

**高频企业法律问题速查索引**（如用户问题匹配以下类型，直接导航至对应法条簇）：

| 问题类型 | 直达法条簇 |
|---------|-----------|
| 违约金是否过高/过低 | 第585条 + 通则解释第64-65条 |
| 合同解除条件 | 第562-564条 + 通则解释第59-60条 |
| 定金罚则 | 第586-588条 |
| 买卖合同质量争议 | 第615-624条 + 买卖合同解释 |
| 租赁合同纠纷 | 第703-734条 + 租赁合同解释 |
| 竞业限制补偿 | 《劳动合同法》第23-24条 + 劳动争议解释(一)第36-40条 |
| 知识产权归属 | 第847-848条、第859-861条 |
| 股东出资纠纷 | 《公司法》第47-54条 + 公司法解释(三) |
| 担保合同效力 | 第681-702条 + 担保制度解释 |
| 个人信息保护 | 《个保法》第13-27条 + 《数据安全法》 |
| 劳动合同解除 | 《劳动合同法》第36-50条 + 劳动争议解释(一) |
| 建设工程纠纷 | 第788-808条 + 建设工程解释(一) |

### 第二步：检索法条及关联规定

基于内置知识库提供核心法条，标注"[建议核实原文]"提示用户验证。建议用户通过北大法宝、法信等平台核实。

**智能关联检索规则**（查到主法条后，自动关联以下内容）：

| 主法条类型 | 自动关联 |
|-----------|---------|
| 《民法典》合同编 | 合同编通则司法解释 + 《九民纪要》相关条目 |
| 《民法典》侵权编 | 人身损害/知产侵权司法解释 |
| 《民法典》物权编 | 担保制度司法解释 + 物权编司法解释 |
| 《民法典》婚姻家庭编 | 婚姻家庭编司法解释(一) |
| 《公司法》条文 | 2024修订说明 + 公司法司法解释(一)(二)(三)(四)(五) |
| 《劳动合同法》 | 实施条例 + 劳动争议解释(一) + 各地高院审判指引 |
| 《个人信息保护法》 | 《数据安全法》+《网络安全法》相关条文 |
| 《广告法》 | 《互联网广告管理办法》 |
| 《反不正当竞争法》 | 不正当竞争司法解释 + 商业秘密司法解释 |
| 《反垄断法》 | 垄断协议/滥用市场支配地位/经营者集中相关规定 |

**法条时效性校验**：
- 检查法条是否为现行有效版本
- 如已修订，标注修订前后对比（新法vs旧法变化要点）
- 如已废止，明确标注并指引至替代法条

### 第三步：实务解读

对每个检索到的法条，提供三层解读：

**第一层：条文原文**
完整引用法条全文，标注法律名称、条款号、生效日期。

**第二层：要点解读**
- 该条文的核心含义（用通俗语言解释法律术语）
- 适用条件和限制
- 与其他条文的关系（特别法优于一般法、新法优于旧法）

**第三层：实务要点**
- 司法实践中的通常理解和裁判倾向
- 常见误区提示（如"违约金不超过30%"的准确理解——是以实际损失为基础，非合同金额的30%）
- 地区差异提示（如各地对某些问题的裁判口径不同）

**多法条竞合时的适用优先级**：

| 优先级规则 | 适用场景 | 示例 |
|-----------|---------|------|
| 特别法优于一般法 | 同位阶法律竞合 | 《劳动合同法》优于《民法典》合同编关于劳动关系的规定 |
| 新法优于旧法 | 同一法律修订前后 | 2024《公司法》优于2018版 |
| 强制性规定优先 | 合同约定与法律冲突 | 约定排除消费者权益的条款无效 |
| 上位法优于下位法 | 法律vs行政法规vs部门规章 | 法律 > 行政法规 > 部门规章 > 地方性法规 |

### 第四步：整理输出

将检索结果按重要性排序：直接适用的法条 > 配套司法解释 > 关联法规 > 典型案例。

## 质量标准

- 法条引用必须包含完整法律名称+条款编号+生效日期
- 优先引用现行有效的法律法规，已废止或修订的条文必须标注
- 司法解释引用需注明发布机关和文号
- 不得混淆不同法律部门的规定
- 实务解读必须基于真实的司法实践，不得编造裁判倾向
- 多法条竞合时必须说明适用优先级和理由
- 关联检索结果必须与用户问题直接相关，不堆砌无关法条
- 操作建议必须具体可行，不以"建议咨询律师"作为唯一建议

## 关联技能

- **审查合同** -- 根据查到的法条审查合同中对应条款的合规性
- **起草合同** -- 根据法条要求起草或修改合同条款
- **NDA快筛** -- 查到保密相关法条后可快速筛查NDA
