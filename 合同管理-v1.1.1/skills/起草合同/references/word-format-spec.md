# 合同Word文档排版规范

本文件为"起草合同"技能生成Word文档时的详细排版参数参考。使用 Node.js `docx` 库生成。

## 1. 页面设置

| 参数 | 值 | docx库参数 |
|-----|---|-----------|
| 纸张 | A4（210mm x 297mm） | `width: 11906, height: 16838`（twip） |
| 上边距 | 25mm | `top: 1417` |
| 下边距 | 25mm | `bottom: 1417` |
| 左边距 | 30mm（含5mm装订线） | `left: 1701` |
| 右边距 | 25mm | `right: 1417` |

## 2. 页眉页脚

**页眉：**
- 左侧：公司名称或"保密"标识，小五号宋体（9pt），颜色 `#666666`
- 右侧：合同编号，小五号宋体（9pt），颜色 `#666666`
- 页眉下方加 0.75pt 单实线，距正文 1mm
- 首页可设置不同页眉（仅显示公司名称，不显示编号）

**页脚：**
- 居中显示页码："第 X 页 共 X 页"
- 小五号宋体（9pt），颜色 `#999999`
- docx实现：`PageNumber.CURRENT` + `PageNumber.TOTAL_PAGES`

## 4. 段落格式

| 元素 | 行距 | 段前 | 段后 | 首行缩进 | 对齐 |
|-----|------|------|------|---------|------|
| 合同大标题 | 固定值28pt | 0pt | 12pt | 无 | 居中 |
| 合同编号 | 固定值28pt | 0pt | 24pt | 无 | 居中 |
| 章标题 | 固定值28pt | 24pt | 12pt | 无 | 居中 |
| 条标题 | 固定值28pt | 12pt | 6pt | 无 | 左对齐 |
| 正文段落 | 固定值28pt | 0pt | 0pt | 2字符(28pt) | 两端对齐 |
| 表格单元格 | 固定值20pt | 2pt | 2pt | 无 | 按列定义 |
| 签章区 | 固定值28pt | 6pt | 6pt | 无 | 左对齐 |

**首行缩进实现：** `indent: { firstLine: 560 }`（14pt x 2字符 x 20twip/pt = 560twip）

## 5. 编号体系

采用五级编号，逐级缩进：

| 层级 | 格式 | 示例 | 缩进(左) |
|-----|------|------|---------|
| 一级 | 第X章 | 第一章 总则 | 0mm |
| 二级 | 第X条 | 第一条 合同标的 | 0mm |
| 三级 | X.X | 1.1 产品名称 | 7mm |
| 四级 | (X) | (1) 具体要求 | 14mm |
| 五级 | a) | a) 子项 | 21mm |

如合同较短（10条以内），可省略"章"层级，直接从"第X条"开始。

## 6. 表格样式

| 参数 | 值 |
|-----|---|
| 表格宽度 | 100%页面可用宽度 |
| 外框线 | 1.5pt 实线，颜色 #000000 |
| 内框线 | 0.5pt 实线，颜色 #000000 |
| 表头底色 | #D9E2F3（浅蓝灰） |
| 表头文字 | 黑体(SimHei) 小四号 加粗 居中 |
| 表格正文 | 仿宋(FangSong) 小四号 左对齐 |
| 单元格内边距 | 上下4pt 左右6pt |
| 跨页表头重复 | 启用 `tableHeader: true` |

## 7. 签章页排版

签章页**必须另起一页**（插入分页符），避免签章区"开天窗"（签章悬空在页面顶部无正文）。

**双栏布局（使用无边框表格实现）：**

- 左栏(占45%)：甲方信息
- 中间留白(10%)
- 右栏(占45%)：乙方信息

每栏包含：
- 单位名称：______________
- 法定代表人（或授权代表）：________
- 日期：____年____月____日
- [盖章位置]（预留40mm x 40mm）

签字线：60-80mm长，0.75pt实线。

## 8. 附件排版

- 每个附件**另起一页**（插入分页符）
- 附件标题：小三号黑体(SimHei)加粗居中，格式"附件X：XXX"
- 附件编号顺序：附件一、附件二...与正文引用一致
- 附件正文沿用主合同正文格式（仿宋四号，首行缩进2字符）

## 9. 特殊标记样式

**【待确认：XXX】标注：**
- 文字颜色：红色 `#FF0000`
- 背景高亮：黄色 `highlight: "yellow"`
- 字体加粗
- docx实现：`new TextRun({ text: "【待确认：XXX】", bold: true, color: "FF0000", highlight: "yellow" })`

**格式条款提示（履行第496条提示说明义务）：**
- 加粗 + 下划线
- docx实现：`bold: true, underline: { type: UnderlineType.SINGLE }`

**重点条款标记：**
- 方式一：条款标题后加标识
- 方式二：条款正文加粗
- 方式三：侧边批注说明重点关注原因

## 10. 水印

合同初稿默认添加 "DRAFT 草稿" 对角线水印：
- 字体：黑体(SimHei)，字号：54pt
- 颜色：浅灰 `#C0C0C0`，透明度：50%
- 旋转角度：-45度，位置：居中置于文字下方

## 11. 金额与日期格式

**金额必须同时使用大小写：**
- 格式：人民币壹拾壹万叁仟元整（\113,000.00）
- 大写金额加粗，千位分隔符

**日期统一格式：**
- 正文中：XXXX年XX月XX日（如2026年04月18日）
- 签章页预留格式：____年____月____日

## 12. 合同编号格式

格式：【公司简称-合同类型缩写-YYYY-NNN】

| 合同类型 | 缩写 | 示例 |
|---------|------|------|
| 采购合同 | CG | 【XX-CG-2026-001】 |
| 销售合同 | XS | 【XX-XS-2026-001】 |
| 技术开发 | JS | 【XX-JS-2026-001】 |
| 租赁合同 | ZL | 【XX-ZL-2026-001】 |
| 服务合同 | FW | 【XX-FW-2026-001】 |
| 股权转让 | GQ | 【XX-GQ-2026-001】 |
| 代理经销 | DL | 【XX-DL-2026-001】 |
| 劳务外包 | LW | 【XX-LW-2026-001】 |
