# 投行业务

投行全业务AI助手，覆盖IPO招股书起草、并购重组报告书、债券募集说明书、交易所问询回复、路演推介材料和财务建模六大核心场景。

> **免责声明：** 本插件辅助专业工作流程，不替代专业意见。所有输出应由具备相应资质的专业人士审核后方可用于决策。

## 适用角色

- **投行项目经理** — IPO/并购/债券项目全流程执行与协调
- **保荐代表人** — 注册制审核材料把关与信披质量控制
- **投行分析师** — 底稿准备、财务建模与数据分析
- **债券承销人员** — 募集说明书起草与路演材料准备

## 典型工作流

```
财务建模 ──→ 招股书/债券募集/并购方案 ──→ 问询回复 ──→ 路演材料
  (估值基础)      (申报材料)              (审核反馈)     (发行推介)
```

## Skills

| Skill | 说明 | 典型输入 |
|-------|------|---------|
| 招股书 | 按注册制要求生成IPO招股说明书各章节初稿，适配科创板/创业板/主板/北交所 | 公司信息+拟上市板块 |
| 并购方案 | 按重大资产重组管理办法生成并购重组报告书初稿，含定价分析和业绩承诺 | 交易背景+标的信息 |
| 债券募集 | 按交易所/银行间要求生成募集说明书初稿，含偿债能力专项分析 | 发行人信息+债券品种 |
| 问询回复 | 逐条回复交易所问询函，含事实陈述+合理性论证+同行业对比+核查意见 | 问询函PDF/Word |
| 路演材料 | 生成IPO/债券/并购路演的PPT大纲、逐页演讲稿和Q&A预案 | 项目信息+路演类型 |
| 财务建模 | 构建CAS三表联动预测模型，含DCF估值、可比公司分析和敏感性测试 | 历史财报数据 |

## 快捷命令

| 命令 | 说明 |
|------|------|
| `/招股书` | 输入公司信息与拟上市板块，生成招股说明书各章节初稿 |
| `/并购方案` | 输入交易背景与标的信息，生成并购重组报告书初稿 |
| `/债券募集` | 输入发行人信息与债券品种，生成募集说明书初稿 |
| `/问询回复` | 上传问询函，逐条生成符合监管格式的回复初稿 |
| `/路演材料` | 输入项目信息与路演类型，生成PPT大纲和演讲稿 |
| `/财务建模` | 上传财报或输入公司名，构建三表联动+DCF估值模型 |
