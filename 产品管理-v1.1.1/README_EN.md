# Product Management

An end-to-end product management toolkit covering PRD writing, user story breakdown, competitive analysis, requirement prioritization, user feedback analysis, product metrics review, roadmap updates, and product brainstorming. Each skill includes built-in methodologies, conditional branching, and output templates for delivery-ready results.

> **Disclaimer:** This plugin supports professional workflows but does not replace professional judgment. All outputs should be reviewed by qualified professionals before being used in decision-making.

## How It Works

```
+--------------------------------------------------+
|  ALWAYS (standalone)                              |
|  * PRD writing (type-specific templates)          |
|  * User story breakdown (INVEST + 5 split modes)  |
|  * Competitive analysis (feature matrix + SWOT +  |
|    differentiation strategy)                      |
|  * Requirement prioritization                     |
|    (RICE / ICE / MoSCoW / KANO)                   |
|  * User feedback analysis                         |
|    (classification + sentiment + pain-point rank) |
|  * Product metrics review                         |
|  * Roadmap updates                                |
|  * Product brainstorming                          |
+--------------------------------------------------+
|  SUPERCHARGED (with connectors)                   |
|  + Notion  --> PRDs written directly to your wiki |
|  + Linear  --> Iteration status auto-synced       |
|  + Figma   --> Design files referenced inline     |
+--------------------------------------------------+
```

## Target Audience

- **Product Managers** -- PRD writing, requirement breakdown, prioritization
- **Directors of Product** -- Competitive analysis, product strategy, feedback insights
- **Project Managers** -- Requirement prioritization, release management, sprint planning
- **Operations Managers** -- User feedback analysis, release notes

## Quick Commands

| Command | Description | Typical Input |
|---------|-------------|---------------|
| `/prd-generation` | Generate a structured PRD | "Build a loyalty points marketplace" |
| `/user-story-breakdown` | Break an Epic into User Stories | "Users can pay for a membership via credit card" |
| `/competitive-analysis` | Multi-dimensional competitor comparison | "Compare Slack, Teams, and Discord" |
| `/requirement-prioritization` | Prioritize the backlog | "1. Reviews 2. Recommendations 3. Export" |
| `/user-feedback-analysis` | User feedback insight report | Upload a feedback spreadsheet |
| `/product-metrics-review` | Review product metrics | "Analyze last week's DAU drop" |
| `/roadmap-update` | Update the product roadmap | "Summarize this sprint's progress" |
| `/product-brainstorm` | Structured ideation session | "How to improve new-user D1 retention" |

## Skills

| Skill | Description | Core Methodology |
|-------|-------------|------------------|
| PRD Generation | Type-specific PRD templates | Product-type branching + 3-step feature decomposition + quality checklist |
| User Story Breakdown | Split Epics into User Stories | INVEST validation + 5 split modes + Story Point estimation |
| Competitive Analysis | Multi-dimensional competitor comparison | Feature matrix + SWOT + 3-tier differentiation strategy |
| Requirement Prioritization | Prioritize the backlog | RICE / ICE / MoSCoW / KANO + framework selection decision tree |
| User Feedback Analysis | Classify feedback and extract insights | 6-category taxonomy + pain-point priority model |
| Product Metrics Review | Systematic metrics review | North Star decomposition + retention + funnels + A/B interpretation |
| Roadmap Update | Roadmap status and planning | Now/Next/Later view + milestone tracking + health score |
| Product Brainstorm | Structured ideation | SCAMPER + 5 Whys + constraint innovation + Impact/Effort matrix |

## MCP Enhancements

This plugin works fully without any MCP connections. Connecting the following MCPs unlocks additional capabilities:

| Connector | Configured Service | Enhancement |
|-----------|--------------------|-------------|
| Docs | Notion | Write PRDs and reports directly to the knowledge base; search existing docs for reference |
| Project Management | Linear | Pull Project/Cycle status and Issue progress for sprint reports and roadmap updates |
| Design | Figma | Reference design file links and specs for PRD generation and competitive UX comparisons |
