# 企业财税

企业财务和税务管理工具，覆盖财务分析、记账凭证、预算分析、增值税管理、汇算清缴、内部审计、财务报表编制、月结管理和凭证参考九大场景。

> **免责声明：** 本插件辅助专业工作流程，不替代专业意见。所有输出应由具备相应资质的专业人士审核后方可用于决策。

## 适用角色

- **会计** — 日常记账、月末关账和报表编制
- **财务经理** — 财务分析、预算管理和报表审核
- **税务经理** — 增值税管理和汇算清缴
- **审计经理** — 内部审计程序和底稿（含SOX/IFRS）

## 快捷命令

| 命令 | 说明 |
|------|------|
| `/财务分析` | 上传报表，输出杜邦分析+偿债+运营+盈利+现金流+Z-Score+EVA全面诊断 |
| `/做凭证` | 上传银行流水/单据，按CAS准则自动生成记账凭证（含外币/研发/年末结转） |
| `/预算分析` | 上传预算和实际数据，输出差异分解、归因和改进建议（含弹性预算/标准成本差异） |
| `/增值税管理` | 上传进销项数据，输出税负测算、异常预警和申报底稿（含出口退税/加计抵减） |
| `/汇算清缴` | 上传科目余额表，输出纳税调整明细和申报底稿（含股权激励/资产损失） |
| `/内部审计` | 提供审计范围，输出审计程序、工作底稿和发现报告（含数据分析审计） |
| `/财务报表编制` | 上传科目余额表，按CAS编制三表并验证勾稽关系 |
| `/月结检查清单` | 生成10步月末关账清单，追踪进度和卡点解决 |

## Skills

| Skill | 说明 |
|-------|------|
| 财务分析 | 杜邦分析+偿债+运营+盈利+现金流+Z-Score+EVA全面诊断 |
| 做凭证 | 银行流水/费用单据自动识别业务类型并生成会计分录（含外币/研发） |
| 预算分析 | 预算vs实际差异分解（数量/价格/结构）、弹性预算、标准成本差异 |
| 增值税管理 | 税负测算、进销项匹配、出口退税、加计抵减、申报底稿 |
| 汇算清缴 | 纳税调整计算、扣除限额校验、股权激励/债务重组/境外所得 |
| 内部审计 | 审计程序设计、工作底稿生成、数据分析审计、SOX/IFRS适配 |
| 财务报表编制 | 按CAS编制三表、期间对比、勾稽验证、合并报表、附注框架 |
| 月结检查清单 | 10步关账流程、进度追踪、制造业成本核算、年度关账补充 |
| 凭证编制参考 | （内部参考）标准分录模板库和编制规范 |
