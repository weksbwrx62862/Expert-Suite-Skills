# Corporate Finance & Tax

Corporate finance and tax management toolkit covering nine scenarios: financial analysis, journal entries, budget analysis, VAT management, annual CIT settlement, internal audit, financial statement preparation, month-end closing, and journal entry reference.

> **Disclaimer:** This plugin assists professional workflows and does not replace professional advice. All outputs should be reviewed by qualified professionals before being used for decision-making.

## Target Roles

- **Accountant** -- Day-to-day bookkeeping, month-end closing, and statement preparation
- **Finance Manager** -- Financial analysis, budget management, and statement review
- **Tax Manager** -- VAT management and annual CIT settlement
- **Audit Manager** -- Internal audit programs and working papers (including SOX/IFRS)

## Quick Commands

| Command | Description |
|---------|-------------|
| `/financial-analysis` | Upload statements for a full diagnostic: DuPont + solvency + efficiency + profitability + cash flow + Z-Score + EVA |
| `/journal-entries` | Upload bank statements or source documents to auto-generate journal entries per CAS (including foreign currency / R&D / year-end closing) |
| `/budget-analysis` | Upload budget and actuals for variance decomposition, attribution, and recommendations (including flexible budget / standard cost variances) |
| `/vat-management` | Upload input/output VAT data for tax burden estimation, anomaly alerts, and filing worksheets (including export VAT refund / incremental input tax credit) |
| `/annual-tax-settlement` | Upload a trial balance for tax adjustment details and filing worksheets (including share-based compensation / asset losses) |
| `/internal-audit` | Provide the audit scope to generate audit programs, working papers, and findings reports (including data analytics auditing) |
| `/financial-statements` | Upload a trial balance to prepare the three core statements per CAS with cross-statement validation |
| `/month-end-closing` | Generate a 10-step month-end closing checklist with progress tracking and blocker resolution |

## Skills

| Skill | Description |
|-------|-------------|
| Financial Analysis | DuPont + solvency + efficiency + profitability + cash flow + Z-Score + EVA full diagnostic |
| Journal Entries | Auto-detect transaction types from bank statements and source documents and generate accounting entries (including foreign currency / R&D) |
| Budget Analysis | Budget vs. actual variance decomposition (volume / price / mix), flexible budget, standard cost variances |
| VAT Management | Tax burden estimation, input/output matching, export VAT refund, incremental input tax credit, filing worksheets |
| Annual CIT Settlement | Tax adjustment calculations, deduction limit checks, share-based compensation / debt restructuring / overseas income |
| Internal Audit | Audit program design, working paper generation, data analytics auditing, SOX/IFRS adaptation |
| Financial Statements | Prepare three core statements per CAS, period-over-period comparison, cross-statement validation, consolidation, notes framework |
| Month-End Closing | 10-step closing process, progress tracking, manufacturing cost accounting, year-end closing supplements |
| Journal Entry Reference | (Internal reference) Standard entry template library and preparation guidelines |
