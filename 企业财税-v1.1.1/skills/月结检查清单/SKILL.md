# /月结检查清单 -- 月末关账流程管理

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

生成标准月末关账检查清单（10步流程），追踪关账进度，提供每步常见问题速查和解决方案。支持制造业成本核算扩展和年度关账补充事项。

**重要提示**：关账清单为标准流程参考，各企业可根据自身业务复杂度调整步骤。特殊事项（如合并报表抵消、外币折算）需根据实际情况补充。

## 能力层级

**独立能力（无需连接器）**：
- 输出完整10步关账检查清单
- 生成进度追踪表（含负责人/截止时间/状态）
- 提供每步常见卡点和解决方案
- 生成标准月结日历和时间线
- 制造业扩展（生产成本核算步骤）
- 年度关账补充事项清单

**增强能力（连接器加持）**：
- 连接 ~~Notion → 关账进度表写入 Notion 数据库，实时追踪各步骤完成状态

## 关账任务清单（10步标准流程）

### 第1步：截止日确认

**目标**：明确各类业务的入账截止时间，确保不遗漏、不跨期。

**检查要点**：
- 业务截止：本月最后一笔销售/采购业务是否已全部入账
- 费用截止：员工报销提交截止时间（通常次月第2个工作日前）
- 发票截止：本月收到的进项发票是否全部认证和入账
- 银行截止：确认最后一笔银行交易日期

**交付物**：截止通知（发给业务部门）+ 截止确认表

### 第2步：银行对账

**目标**：确保银行存款账面余额与银行对账单一致。

**检查要点**：
- 获取所有银行账户的当月对账单
- 编制银行余额调节表
- 逐笔核对未达账项
- 上月未达账项本月是否已清理

**银行余额调节表格式**：
```
银行对账单余额：          [X]
加：企业已收银行未收      [X]
减：企业已付银行未付      [X]
调节后银行余额：          [X]

企业账面余额：            [X]
加：银行已收企业未收      [X]
减：银行已付企业未付      [X]
调节后企业余额：          [X]

差额：                    0
```

**交付物**：银行余额调节表（每个账户一份）

### 第3步：往来对账

**目标**：确认应收/应付账款余额准确，计提坏账准备。

**检查要点**：
- 应收账款：与主要客户对账确认余额
- 应付账款：与主要供应商对账确认余额
- 账龄分析：按30/60/90/180/360天及以上分层统计
- 坏账计提：按账龄或预期信用损失模型计提

**账龄分析和坏账计提**：

| 账龄 | 计提比例参考 | 操作要求 |
|------|------------|---------|
| 1年以内 | 5% | 正常催收 |
| 1-2年 | 10% | 发催收函 |
| 2-3年 | 30% | 评估可回收性 |
| 3年以上 | 50%-100% | 考虑核销 |

**交付物**：往来对账确认表 + 账龄分析表 + 坏账计提计算表

### 第4步：存货盘点确认

**目标**：确认存货账面数与实物一致，处理差异。

**检查要点**：
- 账实核对：本月是否进行了盘点（至少抽盘）
- 暂估入库：已到货未到票的暂估是否及时清理
- 跌价准备：存货是否有减值迹象（滞销/过期/市价下跌）
- 成本结转：已销售商品是否正确结转成本

**交付物**：盘点差异报告 + 暂估清理表 + 存货跌价测试表

### 第4.5步：生产成本核算（制造业专用）

> 仅制造业企业需要执行此步骤。

**目标**：正确归集和分配当月生产成本。

**检查要点**：
- 直接材料：领料单与生产工单匹配，材料消耗准确归集
- 直接人工：生产工人工时统计，按工单分配人工成本
- 制造费用：归集当月制造费用（车间折旧/水电/间接人工/物料消耗）
- 制造费用分配：按工时/机器小时/直接人工比例分配至各产品
- 完工入库：已完工产品办理入库手续，结转完工产品成本
- 在产品：确定在产品约当产量，分配本月生产费用

**成本分配公式**：
```
完工产品成本 = 期初在产品 + 本期投入 - 期末在产品
制造费用分配率 = 制造费用总额 / 分配标准总量
某产品应分配 = 该产品实际标准量 x 分配率
```

**交付物**：成本计算单 + 制造费用分配表 + 完工产品入库汇总

### 第5步：固定资产折旧和无形资产摊销

**目标**：确保本月折旧和摊销已正确计提。

**检查要点**：
- 本月新增固定资产是否已录入并开始计提折旧（次月起提）
- 本月处置/报废资产是否已停止折旧
- 折旧方法和年限是否与会计政策一致
- 无形资产摊销是否按月计提
- 长期待摊费用是否按月分摊

**交付物**：折旧计提明细表 + 摊销计提明细表

### 第6步：预提和待摊处理

**目标**：正确处理权责发生制下跨期费用。

**检查要点**：
- 预提费用：本月已发生但尚未支付（如水电费/利息/租金）
- 待摊费用：预付的跨期费用按月分摊（如年度保险费/软件年费）
- 工资计提：本月工资已计提（含社保公积金企业部分）
- 奖金计提：季度/年度奖金按月预提

**交付物**：预提费用明细表 + 待摊费用摊销表

### 第7步：税金计提

**目标**：正确计提和结转本月应交税金。

**检查要点**：
- 增值税结转：将"应交增值税"各明细余额结转至"未交增值税"
- 附加税计提：城建税、教育费附加、地方教育费附加
- 企业所得税预缴：按季度预缴比例计算
- 印花税：检查本月应税合同
- 房产税/土地使用税：按月/季计提

**交付物**：税金计提计算表 + 增值税结转凭证

### 第8步：跨期调整

**目标**：确保收入和费用归属正确的会计期间。

**检查要点**：
- 收入跨期：本月已发货/已提供服务但未开票的收入是否确认
- 费用跨期：属于本月但下月才收到发票的费用是否暂估入账
- 上月暂估冲回：上月暂估本月收到正式单据后是否红冲重做
- 提前确认：检查是否有将下月收入提前确认的情况

**交付物**：跨期调整明细表

### 第9步：结转损益

**目标**：将所有损益类科目余额结转至"本年利润"。

**检查要点**：
- 收入类科目结转：转入本年利润贷方
- 成本费用类结转：转入本年利润借方
- 结转后检查：所有损益类科目余额应为零
- 本年利润：确认本月净利润/净亏损金额

**交付物**：损益结转凭证

### 第10步：试算平衡

**目标**：验证全部账务处理完成后借贷平衡。

**检查要点**：
- 验证：借方余额合计 = 贷方余额合计
- 验证：借方发生额合计 = 贷方发生额合计
- 检查异常余额方向（如资产类出现贷方余额）
- 确认损益类科目余额全部为零

**异常余额方向速查**：

| 科目性质 | 正常方向 | 异常方向可能原因 |
|---------|---------|----------------|
| 应收账款(贷方) | 借方 | 预收款未转/多收退款未处理 |
| 应付账款(借方) | 贷方 | 预付款未转/多付供应商 |
| 累计折旧(借方) | 贷方 | 冲销过多/录入错误 |
| 应交税费(借方) | 贷方 | 预缴税款大于应交/留抵税额 |

**交付物**：科目余额表（结账后）+ 试算平衡确认

## 年度关账补充事项（12月专用）

除标准10步外，年度关账需额外完成：

| 补充事项 | 操作内容 | 注意点 |
|---------|---------|--------|
| **年度资产减值测试** | 固定资产/商誉/长期股权投资减值测试 | 减值一经确认不得转回（CAS） |
| **递延所得税调整** | 确认/调整递延所得税资产和负债 | 暂时性差异逐项复核 |
| **利润分配** | 提取盈余公积、宣告分配股利 | 须有股东会决议 |
| **所得税汇算准备** | 纳税调整初步测算 | 为次年汇算清缴做准备 |
| **关联交易核对** | 确认关联方交易完整披露 | 上市公司须额外披露 |
| **或有事项评估** | 未决诉讼/担保/承诺事项 | 评估是否需要确认预计负债 |

## 关账时间线

**标准5天月结日历**（T=月末最后一天）：

| 时间 | 完成步骤 | 关键动作 |
|------|---------|---------|
| **T日** | 第1步 | 发出截止通知，确认最后交易 |
| **T+1** | 第2-3步 | 完成银行对账和往来对账 |
| **T+2** | 第4-6步 | 存货确认、折旧摊销、预提待摊 |
| **T+3** | 第7-8步 | 税金计提、跨期调整 |
| **T+4** | 第9-10步 | 结转损益、试算平衡 |
| **T+5** | 出报表 | 编制财务报表（可调用 /财务报表编制） |

**加速关账（3天）**：
- T+1：完成第1-6步（前提：银行对账和折旧可自动化）
- T+2：完成第7-9步
- T+3：第10步 + 出报表

## 常见问题速查

| 步骤 | 常见卡点 | 解决方案 |
|------|---------|---------|
| 银行对账 | 未达账项长期未清理 | 超60天逐笔查明原因，无法收回的做坏账 |
| 往来对账 | 客户不回复对账函 | 电话催促+内部核对发货记录 |
| 存货盘点 | 账实差异大 | 盘盈冲减管理费用，盘亏报批后转营业外支出 |
| 暂估入库 | 暂估长期未清理 | 督促采购催收发票，超3个月评估供应商履约 |
| 折旧计提 | 新增资产未及时入账 | 与资产管理员确认验收单，补录卡片 |
| 预提费用 | 金额无法准确估计 | 参照上月实际或合同约定预提 |
| 增值税结转 | 留抵税额处理 | 有留抵不做结转，留在借方 |
| 跨期调整 | 分不清该计入哪个月 | 按权责发生制：服务/商品实际交付月份确认 |

## 质量标准

- 10步流程须按顺序执行，不可跳步（后步依赖前步结果）
- 每步须有明确的交付物和完成标准
- 银行余额调节表必须调平（差额=0）
- 试算平衡必须通过后方可视为关账完成
- 制造业须完成成本核算后方可结转损益

## 红线规则

1. **绝不跳过试算平衡验证**：即使时间紧迫，第10步必须执行，借贷不平衡意味着存在错误
2. **绝不忽略未达账项的清理**：长期未达账项可能掩盖真实的资金风险
3. **绝不在损益未结转前编制报表**：结转损益是报表编制的前提条件
4. **绝不将跨期费用简单忽略**：权责发生制是会计核算的基本原则，跨期必须调整
5. **绝不在制造业跳过成本核算直接结转**：生产成本未正确归集分配会导致利润严重失真
