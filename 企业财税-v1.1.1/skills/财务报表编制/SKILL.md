# /财务报表编制 -- 按CAS编制财务报表

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

根据科目余额表按企业会计准则（CAS）编制资产负债表、利润表和现金流量表，并进行期间对比分析、勾稽验证和附注框架生成。

**重要提示**：报表编制结果需财务负责人审核确认。涉及重大会计估计和判断的科目（如坏账准备、减值准备、递延所得税）需人工复核。

## 能力层级

**独立能力（无需连接器）**：
- 从科目余额表编制资产负债表、利润表、现金流量表
- 自动进行期间对比（同比/环比）并标注重大变动
- 生成主要科目变动说明
- 三表间勾稽关系验证
- 合并报表基本抵消分录指引
- 财务报表附注框架

**增强能力（连接器加持）**：
- 连接 ~~Notion → 将编制的财务报表写入 Notion 页面

## 连接器（可选增强）

| 连接器 | 增强能力 |
|--------|---------|
| **Notion** | 将编制的财务报表写入 Notion 页面，支持团队在线查阅和归档 |

> 没有连接器也完全可以使用——上传科目余额表 Excel/PDF 即可完成编制。

## 调用方式

```
/财务报表编制 <科目余额表或编制需求>
```

## 输入要求

确认：
1. **数据来源**：科目余额表 / 试算平衡表 / 仅利润数据
2. **报表期间**：月度 / 季度 / 年度（如"2024年12月"）
3. **对比需求**：同比 / 环比 / 预算对比
4. **会计准则**：企业会计准则 / 小企业会计准则
5. **特殊要求**（如有）：合并报表 / 附注编制 / 特定格式

> **输入分支**：
> - **科目余额表（最完整）** → 完整编制三表 + 勾稽验证
> - **试算平衡表** → 编制资产负债表和利润表（现金流量表需补充数据）
> - **只有利润数据** → 仅编制利润表并做变动分析
> - **合并报表** → 增加抵消分录指引
> - **需要附注** → 生成附注框架

## 执行流程

### 第一步：数据获取与校验

**数据校验（必做）**：
- 试算平衡检查：借方余额合计 = 贷方余额合计
- 科目完整性：核对是否涵盖常用一级科目
- 异常值检测：余额方向异常的科目

> 如试算不平衡，立即提示用户检查数据，不继续编制。

### 第二步：编制资产负债表

**科目映射规则（企业会计准则）**：

| 报表项目 | 科目余额取数规则 |
|---------|----------------|
| 货币资金 | 库存现金 + 银行存款 + 其他货币资金 |
| 应收账款 | 应收账款借方余额 - 坏账准备(应收部分) |
| 预付款项 | 预付账款借方余额 + 应付账款借方余额 |
| 存货 | 原材料+在产品+库存商品+发出商品-存货跌价准备 |
| 合同资产 | 合同资产借方余额 - 合同资产减值准备 |
| 固定资产 | 固定资产原值 - 累计折旧 - 固定资产减值准备 |
| 使用权资产 | 使用权资产 - 使用权资产累计折旧 - 减值准备 |
| 无形资产 | 无形资产原值 - 累计摊销 - 无形资产减值准备 |
| 应付账款 | 应付账款贷方余额 + 预付账款贷方余额 |
| 预收款项/合同负债 | 合同负债(企业准则) / 预收账款(小企业准则) |
| 未分配利润 | 期初未分配利润 + 本期净利润 - 提取盈余公积 - 分配利润 |

**企业会计准则 vs 小企业会计准则差异**：

| 报表项目 | 企业会计准则 | 小企业会计准则 |
|---------|------------|--------------|
| 预收款项 | 合同负债 | 预收账款 |
| 减值损失 | 信用减值损失 + 资产减值损失 | 资产减值损失（直接核销法） |
| 租赁 | 使用权资产 + 租赁负债 | 不确认使用权资产 |
| 其他收益 | 有（政府补助） | 无（计入营业外收入） |
| 合同资产 | 有 | 无 |
| 公允价值计量 | 有 | 基本不涉及 |

**恒等式验证**：资产合计 = 负债合计 + 所有者权益合计（必须严格相等）

### 第三步：编制利润表

**CAS格式关键计算逻辑**：
- 营业利润 = 营业收入 - 营业成本 - 税金及附加 - 销售/管理/研发/财务费用 + 其他收益 + 投资收益 + 公允价值变动 - 信用减值损失 - 资产减值损失 + 资产处置收益
- 利润总额 = 营业利润 + 营业外收入 - 营业外支出
- 净利润 = 利润总额 - 所得税费用

**非经常性损益识别**：资产处置损益、非经常性政府补助、非货币性资产交换损益、委托投资损益等需单独标注。

### 第四步：编制现金流量表

**直接法核心项目编制技巧**：

| 项目 | 编制公式（工作底稿法） |
|------|---------------------|
| 销售商品收到现金 | 营业收入+销项税+应收减少+预收增加-坏账增加-票据背书转让 |
| 购买商品支付现金 | 营业成本+进项税+存货增加+应付减少+预付增加 |
| 支付给职工的现金 | 应付职工薪酬贷方发生额-应付增加额 |
| 支付的各项税费 | 应交税费贷方发生额-应交增加额+附加税+所得税实缴 |

**间接法校验**：
- 净利润
- 加：资产减值准备、折旧摊销、处置损益、财务费用（利息部分）
- 加：递延所得税变动
- 加：经营性应收项目减少（增加为负）
- 加：经营性应付项目增加（减少为负）
- = 经营活动现金流量净额

> 间接法结果必须与直接法经营活动现金流净额一致。

### 第五步：期间对比分析

**重大变动标注规则**（满足任一即标注）：
- 变动比例 > 20%
- 变动金额 > 总资产的 5%
- 变动金额 > 营业收入的 10%（利润表科目）

**变动说明模板**：
> [科目名称]本期余额[X]万元，较上期[增加/减少][Y]万元，变动率[Z]%。主要原因：[推断原因]。

**常见变动原因推断逻辑**：
- 应收大增 + 收入增加 → 销售规模扩大
- 存货大增 + 成本未同步 → 备货增加/周转放缓
- 固定资产大增 → 新购设备/在建工程转固
- 短期借款大增 → 新增银行借款满足经营需求

### 第六步：勾稽关系验证

| 验证项 | 勾稽关系 | 容差 |
|--------|---------|------|
| 资产负债表平衡 | 资产 = 负债 + 所有者权益 | 0 |
| 未分配利润衔接 | 期末 = 期初 + 净利润 - 盈余公积 - 分配 | 0 |
| 净利润一致 | 利润表净利润 = 资产负债表未分配利润变动（调整后） | 0 |
| 现金流量表平衡 | 现金净增加额 = 期末货币资金 - 期初货币资金 | 0 |
| 间接法校验 | 间接法经营净额 = 直接法经营净额 | 0 |

### 第七步：合并报表抵消（如适用）

**基本抵消分录类型**：

| 抵消类型 | 抵消分录逻辑 |
|---------|-------------|
| 长期股权投资与子公司所有者权益 | 借：子公司实收资本/资本公积/盈余公积/未分配利润；贷：长期股权投资+少数股东权益 |
| 内部往来 | 借：应付账款(子公司)；贷：应收账款(母公司) |
| 内部交易（未实现利润） | 借：营业收入；贷：营业成本+存货(未实现部分) |
| 内部固定资产交易 | 借：营业收入/营业外收入；贷：固定资产(未实现部分)+累计折旧(多提部分) |

> 合并报表抵消分录较为复杂，以上仅为基本框架。涉及多层持股、交叉持股需专业判断。

### 第八步：财务报表附注框架（如适用）

| 附注章节 | 主要内容 |
|---------|---------|
| 一、公司基本情况 | 注册地/主营业务/报表批准报出日 |
| 二、重要会计政策 | 编制基础/合并范围/收入确认/折旧政策/减值政策 |
| 三、重要会计估计 | 坏账计提比例/折旧年限/公允价值确定方法 |
| 四、税项 | 增值税/企业所得税/享受的税收优惠 |
| 五、重要科目明细 | 应收账款明细及账龄/固定资产明细/长期借款明细 |
| 六、关联方及关联交易 | 关联方清单/交易类型/定价方式/交易金额 |
| 七、或有事项 | 未决诉讼/担保/承诺 |
| 八、资产负债表日后事项 | 利润分配/重大投资/重大诉讼进展 |

## 质量标准

- 资产负债表必须严格平衡
- 利润表各层级利润计算逻辑正确
- 现金流量表间接法和直接法必须一致
- 科目归类符合所选会计准则
- 备抵科目须正确冲减对应资产
- 重大变动标注须覆盖所有超阈值科目

## 红线规则

1. **绝不编造不存在的科目数据**：缺失数据须标注"数据缺失"而非填充零或估算值
2. **资产=负债+所有者权益必须恒等**：不等时须立即停止并报告差异，不得强行调平
3. **现金净增加额必须等于期末减期初**：现金流量表最终数据必须与资产负债表货币资金变动一致
4. **绝不将备抵科目遗漏**：应收账款/存货/固定资产等须减去对应准备/折旧
5. **绝不混淆企业会计准则和小企业会计准则科目**：合同负债vs预收账款、信用减值损失vs坏账损失必须按所选准则使用
