# /预算分析 -- 预算执行差异分析

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

对企业预算执行情况进行差异分析，分解收入/成本/费用差异的数量因素和价格因素，归因差异原因并提出改进建议。支持弹性预算、滚动预测和制造业标准成本四维差异分析。

**重要提示**：差异计算公式明确，输出可验证。归因分析和改进建议基于数据推导，建议管理层结合业务实际确认。

## 能力层级

**独立能力（无需连接器）**：
- 预算 vs 实际逐项对比（有利/不利标记）
- 差异分解（数量因素 + 价格因素 + 结构因素）
- 弹性预算编制和差异分析
- 制造业标准成本四维差异分析
- 归因分析（四象限框架）
- 预算准确度评分和滚动预测建议
- 零基预算审视

**增强能力（连接器加持）**：
- 连接 ~~Notion → 将分析报告写入 Notion 页面，改进行动项在 Notion 中跟踪管理

## 输入要求

确认：
1. **预算数据**：年度 / 季度 / 月度预算表
2. **实际数据**：对应期间的实际执行数据
3. **分析颗粒度**：按部门 / 项目 / 科目 / 产品线
4. **企业类型**：一般企业 / 制造业（有标准成本）/ 服务业
5. **关注重点**（如有）：某一科目或部门

> **输入分支**：
> - 预算 + 实际数据齐全 → 完整六步差异分析
> - 制造业 + 标准成本数据 → 增加四维差异分析（材料/人工/变动制造费/固定制造费）
> - 仅有实际数据无预算 → 与上期/去年同期对比
> - 仅有预算 → 审视预算假设的合理性（零基审视法）

## 执行流程

### 第一步：数据获取

**如果连接了 ~~Notion：**
1. 从 Notion 数据库读取预算表和实际数据
2. 分析结果可写入 Notion 页面

**如果未连接：**
1. 从用户上传的 Excel/PDF 中提取数据
2. 确认预算和实际数据的科目对齐

### 第二步：预算 vs 实际对比

```
差异金额 = 实际数 - 预算数
差异率 = (实际数 - 预算数) / |预算数| x 100%
```

有利/不利标记：收入超预算为有利(F)，成本费用超预算为不利(U)。

**重要性阈值标准**：

| 科目类型 | 差异率阈值 | 差异金额阈值 |
|---------|-----------|------------|
| 营业收入 | >=5% | >=50万 |
| 主营业务成本 | >=5% | >=30万 |
| 期间费用（单项） | >=10% | >=10万 |
| 净利润 | >=10% | >=20万 |

### 第三步：差异分解

**收入差异分解**：
```
数量差异 = (实际销量 - 预算销量) x 预算单价
价格差异 = (实际单价 - 预算单价) x 实际销量
结构差异 = 各产品实际销量 x 预算单价 x (实际结构比-预算结构比)
完整性校验：数量差异 + 价格差异 + 结构差异 = 总差异
```

**制造业标准成本四维差异分析**：

| 差异类型 | 计算公式 | 责任归属 |
|---------|---------|---------|
| **直接材料价格差异** | (实际价格-标准价格) x 实际用量 | 采购部门 |
| **直接材料用量差异** | (实际用量-标准用量) x 标准价格 | 生产部门 |
| **直接人工工资率差异** | (实际工资率-标准工资率) x 实际工时 | 人力资源 |
| **直接人工效率差异** | (实际工时-标准工时) x 标准工资率 | 生产部门 |
| **变动制造费用开支差异** | (实际费率-标准费率) x 实际工时 | 生产管理 |
| **变动制造费用效率差异** | (实际工时-标准工时) x 标准费率 | 生产部门 |
| **固定制造费用预算差异** | 实际固定费用 - 预算固定费用 | 生产管理 |
| **固定制造费用产量差异** | (预算产量-实际产量) x 标准分配率 | 生产计划 |

### 第四步：弹性预算分析

当实际产量/销量偏离预算 >10% 时，必须使用弹性预算：

```
弹性预算 = 固定成本（按预算） + 变动成本（按实际产量 x 预算单位变动成本）
销量差异 = 弹性预算 - 静态预算
弹性预算差异 = 实际 - 弹性预算
静态预算差异 = 销量差异 + 弹性预算差异
```

**成本性态分类参考**：

| 费用类型 | 固定部分 | 变动部分 |
|---------|---------|---------|
| 直接材料 | 无 | 全部变动 |
| 直接人工 | 基本工资 | 计件/加班工资 |
| 制造费用-折旧 | 全部固定 | 无 |
| 制造费用-水电 | 基本费 | 按产量变动 |
| 销售费用-运费 | 无 | 按销量变动 |
| 管理费用 | 绝大部分固定 | 少量变动 |

### 第五步：差异归因与行为效应

**归因四象限框架**：

| 归因维度 | 典型原因 | 可控性 | 改进方向 |
|---------|---------|--------|---------|
| **市场因素** | 需求变化/竞争加剧/原材料涨价 | 低 | 调整预算假设 |
| **经营因素** | 产能不足/效率下降/质量问题 | 高 | 运营改进 |
| **预算因素** | 假设偏差/编制粗糙/未考虑季节性 | 高 | 优化编制方法 |
| **时间因素** | 季节性波动/项目延期/费用跨期 | 中 | 调整预算节奏 |

**预算行为效应检测**：

| 行为 | 检测信号 | 影响 |
|------|---------|------|
| **沙袋效应** | 预算故意做低，实际轻松超额(>15%)完成 | 资源配置不合理 |
| **冲刺效应** | 期末最后一周收入/费用占比>40% | 数据失真 |
| **花光预算** | 费用执行率95-100%但无对应产出增长 | 预算浪费 |
| **年底压单** | 12月收入显著低于趋势值 | 跨期利润调节 |

> 检测到行为效应时须在报告中标注，供管理层决策参考。

### 第六步：改进建议与滚动预测

**预算准确度评分**：

| 差异率区间 | 评分 | 评价 |
|-----------|------|------|
| <5% | A（优秀） | 预算编制精准 |
| 5-10% | B（良好） | 基本准确 |
| 10-20% | C（一般） | 需改进编制方法 |
| >20% | D（差） | 预算假设需重新审视 |

**滚动预测方法**：
```
全年预测 = 已完成月份实际数 + 剩余月份预测数
剩余月份预测 = 原预算 x 修正系数
修正系数 = 已完成期间实际完成率的加权平均
```

**零基预算审视要点**（当预算编制质量为C/D级时建议）：
- 每项费用从零开始论证必要性（而非在上年基础上加减）
- 区分"必须有"vs"最好有"vs"可有可无"三个层级
- 每项预算须回答：这笔钱不花会怎样？花了能带来什么？

**如果连接了 ~~Notion：**
1. 将各项改进建议写入 Notion 数据库
2. 设置责任人和截止日期

## 质量标准

- 差异计算须准确，有利/不利标记正确
- 差异分解须满足：各因素差异之和 = 总差异
- 弹性预算须正确区分固定成本和变动成本
- 制造业四维差异分析须明确责任归属
- 归因分析须具体到四象限中的某一维度
- 滚动预测须说明修正系数的计算依据
- 行为效应检测须有量化信号支撑

## 红线规则

1. **绝不将所有差异笼统归因为"市场原因"**：须区分可控和不可控因素
2. **绝不遗漏差异分解的完整性校验**：各因素差异之和必须等于总差异
3. **绝不将有利差异默认视为好事**：成本大幅低于预算可能意味着产量不足
4. **绝不在产量偏差大时仅用静态预算评价**：偏差>10%时必须使用弹性预算
5. **绝不将标准成本差异的责任归属混淆**：材料价格差异归采购、用量差异归生产，错误归因会导致考核不公
