# 凭证编制参考

常用会计分录模板库、凭证编制规范和常见错误清单。供「做凭证」技能内部参考使用。

## 标准分录模板库

### 采购类

**采购入库（已到票，一般纳税人）**：
```
借：原材料/库存商品  [不含税金额]
借：应交税费——应交增值税(进项税额)  [税额]
  贷：应付账款  [含税金额]
```

**采购入库（暂估，未到票）**：
```
借：原材料/库存商品  [暂估金额（不含税）]
  贷：应付账款——暂估应付账款  [暂估金额]
注：次月初红字冲回，收到发票后按实际金额入账
```

**采购退货**：
```
借：应付账款  [含税金额]
  贷：原材料/库存商品  [不含税金额]
  贷：应交税费——应交增值税(进项税额转出)  [税额]
```

### 销售类

**确认收入（一般纳税人13%）**：
```
借：应收账款/银行存款  [含税金额]
  贷：主营业务收入  [不含税金额]
  贷：应交税费——应交增值税(销项税额)  [税额]
```

**成本结转**：
```
借：主营业务成本  [成本金额]
  贷：库存商品  [成本金额]
```

**销售退回**：
```
借：主营业务收入  [不含税金额]
借：应交税费——应交增值税(销项税额)  [税额]
  贷：应收账款/银行存款  [含税金额]
同时冲回成本：
借：库存商品  [成本金额]
  贷：主营业务成本  [成本金额]
```

**合同负债（预收款，按CAS 14）**：
```
收到预收款：
借：银行存款  [收款金额]
  贷：合同负债  [收款金额]
确认收入时：
借：合同负债  [对应金额]
  贷：主营业务收入  [不含税]
  贷：应交税费——应交增值税(销项税额)  [税额]
```

### 费用类

**工资计提**：
```
借：管理费用/销售费用/生产成本/研发费用——工资  [应发工资]
  贷：应付职工薪酬——工资  [应发工资]
```

**工资发放**：
```
借：应付职工薪酬——工资  [应发工资]
  贷：银行存款  [实发工资]
  贷：应交税费——应交个人所得税  [代扣个税]
  贷：其他应收款——社保(个人部分)  [个人社保]
  贷：其他应收款——公积金(个人部分)  [个人公积金]
```

**社保公积金计提（企业部分）**：
```
借：管理费用/销售费用/生产成本——社保  [企业社保]
借：管理费用/销售费用/生产成本——公积金  [企业公积金]
  贷：应付职工薪酬——社保(企业部分)  [企业社保]
  贷：应付职工薪酬——公积金(企业部分)  [企业公积金]
```

**社保公积金缴纳**：
```
借：应付职工薪酬——社保(企业部分)  [企业社保]
借：应付职工薪酬——公积金(企业部分)  [企业公积金]
借：其他应收款——社保(个人部分)  [个人社保]
借：其他应收款——公积金(个人部分)  [个人公积金]
  贷：银行存款  [总缴纳额]
```

**差旅报销**：
```
借：管理费用/销售费用——差旅费  [报销金额]
借：应交税费——应交增值税(进项税额)  [可抵扣税额，如有]
  贷：其他应收款——备用金/银行存款  [报销金额+抵扣税额]
注：交通费/住宿费取得专票可抵扣进项（一般纳税人）
```

### 资产类

**固定资产购入**：
```
借：固定资产  [不含税金额]
借：应交税费——应交增值税(进项税额)  [税额]
  贷：银行存款/应付账款  [含税金额]
```

**折旧计提**：
```
借：管理费用/销售费用/制造费用——折旧费  [月折旧额]
  贷：累计折旧  [月折旧额]
月折旧额 = (原值 - 残值) / 使用月数
```

**资产处置（出售）**：
```
转入清理：
借：固定资产清理  [净值]
借：累计折旧  [已提折旧]
借：固定资产减值准备  [已提减值]
  贷：固定资产  [原值]
收到处置款：
借：银行存款  [处置收入]
  贷：固定资产清理  [处置收入]
结转损益：
借/贷：固定资产清理  [差额]
  贷/借：资产处置收益  [差额]
```

**使用权资产（新租赁准则）**：
```
初始确认：
借：使用权资产  [租赁付款额现值]
借：租赁负债——未确认融资费用  [总租金-现值]
  贷：租赁负债——租赁付款额  [总租金]
每期支付租金：
借：租赁负债——租赁付款额  [当期租金]
  贷：银行存款  [当期租金]
确认融资费用：
借：财务费用  [当期利息]
  贷：租赁负债——未确认融资费用  [当期利息]
使用权资产折旧：
借：管理费用/制造费用——折旧费  [月折旧额]
  贷：使用权资产累计折旧  [月折旧额]
```

### 税务类

**增值税——月末结转**：
```
当月应交 > 0（销项 > 进项）：
借：应交税费——应交增值税(转出未交增值税)  [应交额]
  贷：应交税费——未交增值税  [应交额]
当月留抵（进项 > 销项）：
不做结转分录，留抵税额保留在"应交增值税"借方
```

**附加税计提**：
```
借：税金及附加  [附加税合计]
  贷：应交税费——应交城建税  [实缴增值税 x 7%或5%]
  贷：应交税费——应交教育费附加  [实缴增值税 x 3%]
  贷：应交税费——应交地方教育费附加  [实缴增值税 x 2%]
```

**企业所得税计提（季度预缴）**：
```
借：所得税费用  [预缴金额]
  贷：应交税费——应交企业所得税  [预缴金额]
预缴金额 = 利润总额(累计) x 25% - 已预缴金额
```

### 投资类

**长期股权投资（成本法，子公司）**：
```
取得投资：
借：长期股权投资  [初始投资成本]
  贷：银行存款  [实际支付]
收到现金股利：
借：银行存款/应收股利  [宣告分配额]
  贷：投资收益  [分配额]
```

**长期股权投资（权益法，联营/合营）**：
```
取得投资：
借：长期股权投资——投资成本  [初始成本]
  贷：银行存款  [实际支付]
确认损益：
借：长期股权投资——损益调整  [份额]
  贷：投资收益  [被投资方净利润x持股比例]
```

**交易性金融资产**：
```
购入：
借：交易性金融资产——成本  [公允价值]
借：投资收益  [交易费用]
  贷：银行存款  [总支付]
期末公允价值变动（升值）：
借：交易性金融资产——公允价值变动  [差额]
  贷：公允价值变动收益  [差额]
```

### 外币类

**外币交易日确认**：
```
借：银行存款——美元户  [外币x交易日即期汇率]
  贷：应收账款——XX客户(USD)  [外币x交易日即期汇率]
```

**期末外币调汇**：
```
货币性项目（银行存款/应收/应付等）用期末即期汇率调整：
升值（收益）：
借：银行存款——外币户  [调整差额]
  贷：财务费用——汇兑损益  [差额]
贬值（损失）：
借：财务费用——汇兑损益  [差额]
  贷：银行存款——外币户  [调整差额]
注：非货币性项目（固定资产/存货等历史成本计量的）不调汇
```

### 年末结转类

**结转本年利润**：
```
借：本年利润  [全年净利润（盈利时）]
  贷：利润分配——未分配利润  [全年净利润]
（亏损时方向相反）
```

**利润分配**：
```
提取法定盈余公积（净利润x10%，累计达注册资本50%后可不提）：
借：利润分配——提取法定盈余公积
  贷：盈余公积——法定盈余公积
分配现金股利（需股东会决议）：
借：利润分配——应付现金股利
  贷：应付股利
结转利润分配明细：
借：利润分配——未分配利润
  贷：利润分配——提取法定盈余公积
  贷：利润分配——提取任意盈余公积
  贷：利润分配——应付现金股利
```

## 凭证编制规范

### 摘要书写标准

| 业务类型 | 摘要格式 | 示例 |
|---------|---------|------|
| 收款 | 收[客户名称][款项性质] | 收XX公司货款 |
| 付款 | 付[供应商名称][款项性质] | 付XX公司采购款 |
| 计提 | 计提[月份][费用类型] | 计提12月工资 |
| 结转 | 结转[月份][业务内容] | 结转12月损益 |
| 报销 | [部门][人员][费用类型] | 销售部张三差旅费 |
| 调汇 | [月份]外币账户汇兑调整 | 12月美元户调汇 |

### 附件要求

| 凭证类型 | 必附单据 |
|---------|---------|
| 收入凭证 | 销售合同/订单、发票、发货单 |
| 采购凭证 | 采购合同、发票、验收单 |
| 费用报销 | 费用报销单、发票、审批单 |
| 工资凭证 | 工资表（签字版）、银行代发回单 |
| 折旧/摊销 | 折旧/摊销计算表 |
| 税费凭证 | 完税凭证/缴款书 |

### 审批流程

| 凭证金额 | 审批人 |
|---------|--------|
| 5万以下 | 会计主管 |
| 5-20万 | 财务经理 |
| 20万以上 | 财务总监/总经理 |

## 常见错误清单

| 错误类型 | 错误表现 | 正确做法 |
|---------|---------|---------|
| **借贷方向错误** | 费用记贷方、收入记借方 | 费用增加记借方，收入增加记贷方 |
| **科目使用不当** | 管理人员工资计入销售费用 | 按人员所属部门归集费用科目 |
| **进项税处理错误** | 小规模纳税人进项税单独入账 | 小规模纳税人进项税计入成本 |
| **忘记计提** | 月末未计提折旧/工资/社保 | 每月末必须完成所有计提分录 |
| **暂估未冲回** | 收到发票后直接入账不冲暂估 | 先红冲暂估分录，再按发票重新入账 |
| **个税处理错误** | 代扣个税计入管理费用 | 从应付职工薪酬中扣减，贷记应交税费 |
| **混淆增值税明细** | 所有增值税都记"应交增值税" | 必须区分销项/进项/转出未交/已交等明细 |
| **跨期确认** | 1月收到12月发票直接计入1月 | 应计入12月（权责发生制），或12月暂估 |
| **附件缺失** | 凭证无原始单据支撑 | 每笔凭证必须附完整原始凭证 |
| **摘要不清** | 摘要只写"付款""收款" | 须注明对方单位和款项性质 |
| **外币不调汇** | 期末外币货币性项目不调整 | 必须按期末即期汇率调整并确认汇兑损益 |
| **利润分配顺序错** | 先分红后提盈余公积 | 必须先提法定盈余公积再分配股利 |
