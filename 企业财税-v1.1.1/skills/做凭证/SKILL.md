# /做凭证 -- 自动生成记账凭证

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

按企业会计准则（CAS）从银行流水、费用单据等原始凭证自动识别业务类型并生成记账凭证。支持外币业务、研发费用资本化和年末利润分配。

## 能力层级

**独立能力（无需连接器）**：
- 从用户上传的 Excel/PDF 中识别业务并生成凭证
- 支持批量银行流水、逐笔报销单、口述业务场景
- 自动校验借贷平衡，按日期编号
- 外币业务自动套用汇率并计算汇兑损益
- 期末结转（损益结转、利润分配五步链）

**增强能力（连接器加持）**：
- 连接 ~~Notion → 从 Notion 数据库读取凭证模板，生成结果写入 Notion 页面

## 连接器（可选增强）

| 连接器 | 增强能力 |
|--------|---------|
| **Notion** | 管理凭证模板库，生成的凭证写入 Notion 页面方便团队共享 |

> 没有连接器也完全可以使用——上传 Excel/PDF 即可生成凭证。

## 调用方式

```
/做凭证 <银行流水或费用单据>
```

帮我把这些做成凭证：@$1

接受银行流水（Excel/PDF）、费用报销单、发票信息、口述业务描述。

## 输入要求

确认：
1. **数据类型**：银行流水 / 费用报销单 / 发票 / 业务描述
2. **企业会计制度**：企业会计准则 / 小企业会计准则
3. **增值税身份**：一般纳税人 / 小规模纳税人（影响进项税处理）
4. **币种**：人民币 / 涉及外币（需确认记账本位币和汇率来源）
5. **特殊需求**（如有）：辅助核算要求 / 部门分摊 / 项目核算

> **输入分支**：
> - 上传标准银行流水 Excel → 批量识别 + 批量生成凭证
> - 上传费用报销单 → 逐笔识别业务类型 + 生成分录
> - 口述业务场景 → 生成单笔标准分录 + 附准则解释
> - 外币业务 → 生成外币分录 + 期末汇兑损益调整
> - 期末结转需求 → 生成损益结转 / 利润分配完整分录链

## 执行流程

### 第一步：数据获取

**如果连接了 ~~Notion：**
1. 从 Notion 数据库读取凭证模板或历史分录参考
2. 生成的凭证可写入 Notion 页面

**如果未连接：**
1. 从用户上传的 Excel/PDF 中提取数据
2. 如用户口述业务，整理后确认

### 第二步：业务识别

从原始单据中识别业务类型：

| 业务大类 | 细分类型 |
|---------|---------|
| **收入类** | 销售商品 / 提供劳务 / 利息收入 / 投资收益 / 政府补助 / 合同变更 |
| **采购类** | 采购原材料 / 采购商品 / 采购固定资产 / 采购服务 / 暂估入库 |
| **费用类** | 工资薪金 / 社保公积金 / 差旅费 / 办公费 / 租赁费 / 招待费 / 广告费 |
| **资金类** | 银行借款 / 还款付息 / 股东投资 / 利润分配 / 外币兑换 |
| **税费类** | 增值税缴纳 / 企业所得税 / 个人所得税代扣代缴 / 附加税 |
| **期末类** | 折旧摊销 / 减值准备 / 结转损益 / 计提费用 / 汇兑损益 |
| **研发类** | 研发费用化支出 / 研发资本化支出 / 无形资产摊销 |

### 第三步：科目匹配

按 CAS 科目体系匹配会计科目：

**常用科目速查**：
- 银行存款(1002) / 库存现金(1001) / 其他货币资金(1012)
- 应收账款(1122) / 应收票据(1121) / 合同资产(1151)
- 预付账款(1123) / 其他应收款(1221)
- 原材料(1403) / 库存商品(1405)
- 固定资产(1601) / 累计折旧(1602) / 使用权资产(1901)
- 无形资产(1701) / 长期待摊费用(1801) / 研发支出(5301)
- 应付账款(2202) / 应付票据(2201) / 合同负债(2205)
- 应交税费(2221) / 应付职工薪酬(2211)
- 主营业务收入(6001) / 其他业务收入(6051) / 营业外收入(6301)
- 主营业务成本(6401) / 管理费用(6602) / 销售费用(6601) / 研发费用(6604) / 财务费用(6603)

> **准则分支判断**：
> - **企业会计准则** → 使用合同负债(2205)、使用权资产(1901)、合同资产(1151)、信用减值损失(6711)、其他收益(6117)
> - **小企业会计准则** → 使用预收账款(2203)、不确认使用权资产、不计提减值准备、无合同资产科目、无其他收益科目

### 第四步：生成分录

**销售商品（一般纳税人 13%）**：
```
借：应收账款/银行存款  [含税金额]
  贷：主营业务收入  [不含税金额]
  贷：应交税费——应交增值税(销项税额)  [税额]
同时结转成本：
借：主营业务成本  [成本金额]
  贷：库存商品  [成本金额]
```

**销售商品（小规模纳税人）**：
```
借：应收账款/银行存款  [含税金额]
  贷：主营业务收入  [不含税=含税/(1+1%)]
  贷：应交税费——应交增值税  [税额=不含税x1%]
```

**采购原材料（一般纳税人）**：
```
借：原材料  [不含税金额]
借：应交税费——应交增值税(进项税额)  [税额]
  贷：应付账款/银行存款  [含税金额]
```

**暂估入库（未收到发票）**：
```
借：原材料/库存商品  [暂估不含税金额]
  贷：应付账款——暂估应付账款  [暂估金额]
注：次月初红字冲回，收到发票后按实际金额重新入账
```

**计提工资**：
```
借：管理费用/销售费用/生产成本/研发费用  [应付工资]
  贷：应付职工薪酬——工资  [应付工资]
```

**缴纳社保**：
```
借：应付职工薪酬——社保(企业部分)
借：其他应收款——社保(个人部分)
  贷：银行存款
```

**政府补助（与日常活动相关）**：
```
借：银行存款  [补助金额]
  贷：其他收益  [补助金额]
注：与日常活动无关的政府补助计入营业外收入
```

**新租赁准则（承租方初始确认）**：
```
借：使用权资产  [租赁付款额现值]
借：租赁负债——未确认融资费用  [总租金-现值]
  贷：租赁负债——租赁付款额  [总租金]
```

**研发费用（费用化阶段）**：
```
借：研发支出——费用化支出  [研发支出]
  贷：原材料/应付职工薪酬/累计折旧  [对应金额]
期末结转：
借：研发费用  [费用化金额]
  贷：研发支出——费用化支出  [费用化金额]
```

**研发费用（资本化阶段）**：
```
借：研发支出——资本化支出  [研发支出]
  贷：原材料/应付职工薪酬/累计折旧  [对应金额]
达到预定用途时：
借：无形资产  [资本化总额]
  贷：研发支出——资本化支出  [资本化总额]
```

**外币业务（交易日）**：
```
借：银行存款——美元户  [外币金额 x 交易日即期汇率]
  贷：应收账款——XX客户(美元)  [外币金额 x 交易日即期汇率]
```

**外币业务（期末调汇）**：
```
汇兑收益：
借：银行存款——美元户/应收账款(美元)  [调整差额]
  贷：财务费用——汇兑损益  [差额]
汇兑损失：
借：财务费用——汇兑损益  [差额]
  贷：银行存款——美元户/应收账款(美元)  [调整差额]
```

**月末结转损益**：
```
收入类科目结转：
借：主营业务收入/其他业务收入/营业外收入/其他收益...
  贷：本年利润
费用类科目结转：
借：本年利润
  贷：主营业务成本/管理费用/销售费用/财务费用/研发费用/营业外支出...
```

**年末利润分配五步链**：
```
第一步：结转本年利润
借：本年利润  [全年净利润]
  贷：利润分配——未分配利润  [全年净利润]
第二步：提取法定盈余公积（净利润x10%，累计达注册资本50%后可不提）
借：利润分配——提取法定盈余公积  [金额]
  贷：盈余公积——法定盈余公积  [金额]
第三步：提取任意盈余公积（股东会决议比例）
借：利润分配——提取任意盈余公积  [金额]
  贷：盈余公积——任意盈余公积  [金额]
第四步：分配股利
借：利润分配——应付现金股利  [金额]
  贷：应付股利  [金额]
第五步：结转利润分配明细
借：利润分配——未分配利润  [上述合计]
  贷：利润分配——提取法定盈余公积
  贷：利润分配——提取任意盈余公积
  贷：利润分配——应付现金股利
```

### 第五步：凭证编号和汇总

1. 按日期排序编号（记-YYYY-MM-001 起编）
2. 汇总借贷方合计
3. 验证每笔借贷平衡
4. 验证全部凭证借方合计 = 贷方合计

## 质量标准

- 科目使用须符合 CAS 科目体系，新准则科目优先（合同负债/使用权资产/信用减值损失/其他收益）
- 每笔分录须借贷平衡，批量生成后须汇总校验
- 增值税处理须严格区分一般纳税人和小规模纳税人
- 一般纳税人须正确区分销项税额、进项税额、进项转出
- 小规模纳税人进项税计入成本，不单独列示进项
- 费用科目须按性质归类（管理/销售/研发/财务/生产成本）
- 外币业务须注明原币金额、汇率和折算后本位币金额
- 研发费用须区分费用化和资本化阶段，资本化须满足五个条件

## 常见错误清单

| 错误类型 | 正确做法 |
|---------|---------|
| 小规模纳税人进项单独列账 | 进项税直接计入采购成本 |
| 政府补助一律计入营业外收入 | 与日常活动相关的计入其他收益 |
| 使用权资产不计提折旧 | 须按直线法在租赁期内计提折旧 |
| 个税代扣代缴计入管理费用 | 应借记应付职工薪酬，贷记应交税费 |
| 收到银行承兑汇票计入银行存款 | 应计入应收票据 |
| 研发资本化不满足条件就确认 | 须同时满足五个条件方可资本化 |
| 外币账户期末不调汇 | 外币货币性项目期末须按即期汇率调整 |
| 暂估入库含进项税 | 暂估金额为不含税价（一般纳税人） |
| 年末未做利润分配结转 | 须完成利润分配五步链 |

## 红线规则

1. **绝不混淆一般纳税人和小规模纳税人的分录**：一般纳税人进项税单独入账可抵扣；小规模纳税人进项税计入成本，分录逻辑完全不同
2. **绝不遗漏借贷平衡校验**：每笔分录必须借贷相等，批量生成后须汇总校验总借方 = 总贷方
3. **绝不在无法识别业务类型时猜测科目**：遇到模糊业务描述（如"款项""其他""转账"），须向用户确认具体业务性质后再编制分录
4. **绝不将应交税费的明细科目混用**：增值税下设销项税额/进项税额/转出未交增值税/已交税金等，必须准确使用对应明细
5. **绝不将研发资本化条件不满足时强行资本化**：须技术可行性、完成意图、使用/出售能力、未来经济利益、资源支持五项同时满足
