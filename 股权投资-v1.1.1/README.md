# 股权投资

PE/VC股权投资全流程工具，覆盖项目筛选、尽调清单、条款审查、投决备忘录、收益测算和退出分析六大场景。

> **免责声明：** 本插件辅助专业工作流程，不替代专业意见。所有输出应由具备相应资质的专业人士审核后方可用于决策。

## 适用角色

- **投资经理** -- 项目筛选和尽调的核心工具
- **投资总监** -- 投决材料准备和风险把控
- **合伙人** -- 快速评估投资机会
- **投后管理** -- 退出分析和收益跟踪

## 快捷命令

| 命令 | 说明 | 典型输入 |
|------|------|---------|
| `/筛项目` | BP/CIM快速筛选，输出一页六维评分备忘录 | 上传BP或描述项目 |
| `/尽调清单` | 生成结构化DD清单（财务/法律/业务/技术+行业专项） | 项目名称+行业+架构类型 |
| `/审条款` | TS/SPA逐项审查，含九民纪要合规检查和谈判建议 | 上传Term Sheet或SPA |
| `/投决备忘录` | 生成投委会IC Memo，含投资逻辑、估值和条款摘要 | 项目信息+尽调发现 |
| `/测收益` | IRR/MOIC/DPI测算+25格敏感性分析表 | 交易条款+退出假设 |
| `/退出分析` | 五路径退出对比（IPO/并购/S基金/回购），含时间成本估算 | 被投公司现状+基金期限 |

## 典型工作流

```
发现项目 → /筛项目 (快筛) → /尽调清单 (启动DD) → /审条款 (审交易文件)
                                                          ↓
                    /退出分析 (投后管理) ← /测收益 (算回报) ← /投决备忘录 (上IC)
```
