# 中国PE/VC退出渠道参考

## 一、A股各板块上市标准速查

### 上交所主板

| 标准 | 要求 |
|------|------|
| **盈利要求** | 最近3年净利润均为正且累计>1.5亿，或最近1年净利润>1亿且营收>10亿 |
| **现金流/营收** | 最近3年经营活动现金流净额累计>2亿，或最近3年营收累计>15亿 |
| **市值** | 不低于80亿（预计市值） |
| **适合行业** | 传统行业龙头、大型制造企业、金融机构 |

### 科创板

| 标准 | 标准一 | 标准二 | 标准三 | 标准四 | 标准五 |
|------|--------|--------|--------|--------|--------|
| **市值** | >=10亿 | >=15亿 | >=20亿 | >=30亿 | >=40亿 |
| **盈利** | 近2年净利润均正且累计>=5000万 | 近1年营收>=3亿 | 近1年营收>=3亿+研发占比>=15% | 主营>3亿、市场空间大 | 核心产品获国家批准 |
| **适合行业** | 硬科技、芯片、新材料、生物医药、AI |

### 创业板

| 标准 | 要求 |
|------|------|
| **盈利要求** | 最近2年净利润均正且累计>=5000万，或最近1年净利润正且营收>=3亿 |
| **市值** | 不低于10亿（预计市值） |
| **适合行业** | 创新型成长企业、TMT、新消费、新能源 |

### 北交所

| 标准 | 标准一 | 标准二 | 标准三 | 标准四 |
|------|--------|--------|--------|--------|
| **市值** | >=2亿 | >=4亿 | >=8亿 | >=15亿 |
| **财务** | 近2年净利润均正且平均>=1500万 | 近1年营收>=2亿+增速>=30% | 近1年营收>=1亿+市值>=8亿 | 市值>=15亿+研发>=5000万 |
| **适合行业** | 专精特新中小企业 |

> 以上标准可能随政策调整更新，建议以证监会/交易所最新规则为准。

## 二、港股上市标准

### 主板

| 测试 | 要求 |
|------|------|
| **盈利测试** | 最近3年累计盈利>=8000万港元（最近1年>=3500万，前2年合计>=4500万） |
| **市值/收入测试** | 市值>=40亿港元+最近1年收入>=5亿港元 |
| **市值/收入/现金流测试** | 市值>=20亿港元+最近1年收入>=5亿港元+前3年经营活动现金流合计>=1亿港元 |

### 18A章（生物科技）

- 市值>=15亿港元
- 至少有1种核心产品已通过概念阶段
- 营运资金足够覆盖至少12个月+上市募资金额的125%

## 三、各退出路径成本参考

| 退出方式 | 直接成本 | 时间成本 | 隐性成本 |
|---------|---------|---------|---------|
| A股IPO | 中介费3-5%+其他费500-1000万 | 2-3年 | 限售期流动性损失、信息披露义务 |
| 港股IPO | 中介费4-6%+其他费 | 1-2年 | 流动性折价(小市值)、汇率风险 |
| 并购 | 顾问费1-3% | 6-18个月 | 估值折价、整合失败风险 |
| S基金 | 折扣20-30%(NAV 7-8折) | 3-6个月 | 折价损失 |
| 回购 | 利息补偿(通常年化8%) | 1-3个月(无纠纷) | 执行风险、关系损耗 |

## 四、S基金市场参考

### S基金估值折扣参考

| 项目质量 | 典型折扣率 | 说明 |
|---------|-----------|------|
| 明星项目（已盈利、有明确IPO计划） | NAV的85-95% | 折扣小，抢手 |
| 优质项目（增长良好、退出路径清晰） | NAV的75-85% | 市场主流 |
| 一般项目（增长放缓、退出不确定） | NAV的60-75% | 需时间找买方 |
| 问题项目（亏损、退出困难） | NAV的30-60%或更低 | 可能无人接盘 |

### S基金交易注意事项

1. **优先购买权**：其他股东通常有优先购买权，须提前通知
2. **GP同意**：基金份额转让通常需GP同意
3. **税务安排**：转让定价影响税负，需提前规划
4. **信息披露**：卖方需配合买方尽调，提供基金和底层资产信息
