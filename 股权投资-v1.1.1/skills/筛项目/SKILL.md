# /筛项目 -- PE/VC项目快速筛选

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

**独立能力（无需连接器）**

- BP/CIM关键信息提取 + 红线快筛（一票否决）
- 六维量化评分（每维度1-5分，明确打分锚点）+ 风险矩阵
- 按项目阶段/基金策略差异化评估

> 本技能无需连接器即可完整使用——以上功能会通过手动输入来替代。

快速筛选评估PE/VC项目，从商业计划书（BP）或保密信息备忘录（CIM）中提取关键信息，按投资逻辑框架输出一页筛选备忘录。

**重要提示**：本技能辅助投资团队快速筛选项目，不构成投资建议。项目筛选涉及商业判断，建议投资经理结合实际情况确认。

## 调用方式

```
/筛项目 <BP/CIM文件或项目描述>
```

帮我筛一下这个项目：@$1

## 输入要求

确认：
1. **项目材料**：BP/CIM/项目描述
2. **基金信息**（如有）：人民币基金/美元基金、阶段偏好、行业偏好、单笔投资区间
3. **关注维度**（如有特别要求）

**输入不完整时**：
- 只有项目名称/行业 --> 基于公开信息初步评估，标注"[信息有限，建议补充BP]"
- 有BP无基金信息 --> 按通用标准评估
- 非标材料（新闻/融资公告）--> 提取可得信息，标注信息缺口

## 执行流程

### 第零步：红线快筛（一票否决）

| 红线 | 判断标准 | 处置 |
|------|---------|------|
| 行业政策禁区 | P2P/ICO/课外学科培训/开采类无证矿企 | 直接放弃 |
| 创始人重大负面 | 失信被执行人/刑事案件/行业黑名单/多次创业均失败且无反思 | 直接放弃 |
| 财务造假嫌疑 | 收入/利润与现金流严重背离（经营现金流/净利润<0.3连续2年）、审计非标 | 直接放弃 |
| 核心资质缺失 | 须持牌经营但无牌照且12个月内无法获取 | 直接放弃 |
| 估值严重脱离 | 估值倍数>同行业同阶段中位数3倍以上且无合理解释 | 标记高风险 |

如触发红线，直接输出简要结论，无需继续后续步骤。

### 第一步：关键信息提取

从BP/CIM中提取：

| 维度 | 提取内容 |
|------|---------|
| 基本面 | 公司名称、成立时间、注册地（境内/VIE/红筹）、行业、阶段 |
| 业务模式 | 做什么、卖给谁、怎么赚钱、核心价值主张 |
| 团队 | 创始人背景、核心团队完整性、稳定性 |
| 财务 | 收入规模、增速、毛利率、净利率、burn rate |
| 融资历史 | 历史轮次/金额/估值/投资人 |
| 本轮融资 | 金额、估值(pre/post)、资金用途 |

### 第二步：六维量化评分

每维度1-5分，打分锚点如下：

**赛道评估**（TAM/增速/政策/竞争格局）：

| 分值 | 锚点 |
|------|------|
| 5 | TAM>500亿且CAGR>30%，政策明确鼓励，竞争格局未定 |
| 4 | TAM>200亿且CAGR>20%，政策中性偏正，有明确空间 |
| 3 | TAM>100亿且CAGR>15%，竞争激烈但有差异化空间 |
| 2 | TAM<100亿或CAGR<10%，红海竞争 |
| 1 | 市场萎缩/政策限制/行业天花板明显 |

**商业模式**（盈利清晰度/单位经济/可复制性）：

| 分值 | 锚点 |
|------|------|
| 5 | LTV/CAC>5，毛利>70%，强网络效应/规模效应 |
| 4 | LTV/CAC>3，毛利>50%，模式可复制性强 |
| 3 | LTV/CAC>2，毛利>30%，单位经济模型成立 |
| 2 | LTV/CAC<2或毛利<30%，盈利路径不清晰 |
| 1 | 商业模式未验证/持续烧钱无拐点 |

**竞争壁垒**（技术/品牌/渠道/数据/规模/牌照壁垒）：

| 分值 | 锚点 |
|------|------|
| 5 | 核心专利/独占资源，竞争者复制需>3年 |
| 4 | 明确壁垒（数据积累/品牌认知），复制需1-3年 |
| 3 | 有一定壁垒但非不可逾越，先发优势 |
| 2 | 壁垒薄弱，竞争者6-12个月可追上 |
| 1 | 无明显壁垒，完全可被替代 |

**团队评估**（行业认知/执行力/融资能力/互补性）：

| 分值 | 锚点 |
|------|------|
| 5 | 连续成功创业者/行业顶尖专家，核心岗位齐备，里程碑超额完成 |
| 4 | 行业>5年+成功创业经历，团队互补，执行力强 |
| 3 | 相关行业背景，团队基本齐备，执行中规中矩 |
| 2 | 行业经验不足或团队不完整（缺CTO/缺销售负责人） |
| 1 | 跨行业创业且无相关资源，团队不稳定/核心离职 |

**财务健康度**（增速/毛利趋势/现金安全边际）：

| 分值 | 锚点 |
|------|------|
| 5 | 收入增速>100%（早期）或>50%（成长期），现金跑道>18个月 |
| 4 | 增速>50%（早期）或>30%（成长期），跑道>12个月 |
| 3 | 增速>30%（早期）或>20%（成长期），跑道>9个月 |
| 2 | 增速<20%或跑道<6个月 |
| 1 | 负增长或现金将耗尽（<3个月） |

**估值合理性**（PS/PE倍数 vs 可比）：

| 分值 | 锚点 |
|------|------|
| 5 | 估值<同行业同阶段中位数0.7x |
| 4 | 估值在中位数0.7x-1.0x |
| 3 | 估值在中位数1.0x-1.5x |
| 2 | 估值在中位数1.5x-2.5x |
| 1 | 估值>中位数2.5x且无合理溢价解释 |

### 阶段差异化评估

| 阶段 | 权重侧重 | 可弱化 | 核心判断 |
|------|---------|--------|---------|
| 天使/种子 | 团队x2、赛道x1.5 | 财务、估值 | "对的人做对的事" |
| A轮 | 模式x1.5、壁垒x1.5 | 盈利性 | PMF信号是否存在 |
| B轮 | 财务x1.5、模式x1.5 | — | 增速能否证明可复制 |
| C轮/Pre-IPO | 财务x2、估值x1.5 | 团队（已验证） | 利润率和IPO可行性 |

### 第三步：风险矩阵

按影响程度(高/中/低) x 发生概率(高/中/低)绘制风险矩阵：

| 风险类别 | 识别要点 |
|---------|---------|
| 业务风险 | 竞争加剧、技术路线变更、大客户流失（>30%单一客户）、政策收紧 |
| 财务风险 | 持续亏损无拐点、现金流紧张、收入确认激进 |
| 治理风险 | VIE合规、核心团队稳定性、关联交易、代持未清理 |
| 退出风险 | IPO路径不清晰、行业并购不活跃、回购主体无支付能力 |

**高频遗漏风险自查**：
- 社保公积金是否足额缴纳？（中国PE高频踩雷）
- 知识产权归属公司还是个人？
- 未披露的对外担保/质押？
- VIE架构：外商投资负面清单覆盖否？

### 第四步：筛选结论

| 结论 | 条件 |
|------|------|
| 推荐深入 | 加权均分>3.5，无红线，投资逻辑清晰 |
| 有条件推荐 | 均分3.0-3.5，有可解决的风险点 |
| 暂时搁置 | 均分<3.0但赛道或团队突出，建议跟踪 |
| 建议放弃 | 触发红线/核心逻辑不成立 |

如有基金信息，增加匹配度判断：阶段匹配/行业匹配/规模匹配/DPI要求匹配。

## 质量自检

- [ ] 每个评分维度有数据或事实支撑（不允许无依据打分）
- [ ] 估值对比选择了合理可比对象（同行业同阶段）
- [ ] 风险清单覆盖业务/财务/治理/退出四类
- [ ] 红线快筛已执行并明确标注结果
- [ ] 阶段差异化权重已应用
- [ ] 所有判断标注了依据和信息局限性
