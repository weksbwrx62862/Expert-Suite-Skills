# /审条款 -- 投资条款审查

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

**独立能力（无需连接器）**

- 逐项条款提取+四类归档（经济性/控制性/退出/其他）
- 九民纪要合规检查（对赌/回购/担保/让与担保）
- 按投资方/被投方视角差异化谈判建议+优先级排序

> 本技能无需连接器即可完整使用，所有功能基于用户提供的文件和文本。

**重要提示**：条款审查结果仅供参考，最终须由专业律师出具法律意见。

## 调用方式

```
/审条款 <TS/SPA文件>
```

帮我审一下这个条款：@$1

## 输入要求

确认：
1. **文件类型**：Term Sheet / SPA / SHA / 补充协议
2. **审查视角**：投资方 / 被投方（创始人）/ 双方平衡
3. **架构类型**：境内 / VIE（涉及境外文件）
4. **轮次阶段**：天使/A轮/B轮/C轮及以后/Pre-IPO
5. **特别关注条款**（如有）

**轮次差异**：
- 天使/A轮 --> 侧重创始人保护（反稀释、竞业、控制权）
- B轮+ --> 关注老股东权利兼容、多层清算偏好堆叠
- Pre-IPO --> 重点审查对赌可执行性、上市承诺条款

## 执行流程

### 第一步：条款逐项提取

从文件中提取关键条款，归入四类：

**经济性条款**：
- 投资金额和估值（pre/post-money）
- 优先清算权（参与型/非参与型，倍数，cap）
- 反稀释（完全棘轮/加权平均，广义/窄义）
- 优先分红权、转换权

**控制性条款**：
- 董事会构成和席位
- 保护性条款（一票否决事项）
- 信息权和检查权
- 重大事项审批清单

**退出相关条款**：
- 对赌（业绩/上市，金额/期限/补偿方式）
- 回购权（触发/价格/主体/资金来源）
- 共售权、领售权、随售权、ROFR、锁定期

**其他条款**：
- 竞业禁止、陈述保证、违约责任、争议解决

**条款遗漏检查**：提取完成后核对清单，缺少项标注"[原文件未约定，建议补充]"。

### 第二步：条款评估（对照市场标准）

**关键条款市场标准速查**：

| 条款 | 市场标准 | 偏投资方信号 | 偏创始人信号 |
|------|---------|------------|------------|
| 优先清算权 | 1x非参与型 | >1x或参与型无cap | 无清算权 |
| 反稀释 | 广义加权平均 | 完全棘轮 | 无反稀释 |
| 业绩对赌 | 3年期，现金+股权补偿 | >3年/仅现金/无上限 | 无对赌 |
| 竞业禁止 | 2年以内+合理补偿 | >2年或无补偿 | 无竞业 |
| 回购利率 | 年化8-10%单利 | >12%复利 | 无回购 |
| 董事会 | 投资方1席 | 多数席/一票否决 | 创始人多数 |
| 领售权 | 有最低价格保护 | 无最低价格 | 无领售权 |
| 信息权 | 季度财报+年度审计 | 月报+随时检查 | 仅年报 |

**风险等级判定**：
- **高风险** -- 明显偏离市场标准，严重不利于审查方
- **中风险** -- 略偏离，有谈判空间
- **低风险** -- 符合标准或有利于审查方

**交叉验证**：
- 投资金额/估值/持股% 三者自洽（持股% = 投资额/post-money）
- 多轮清算偏好叠加后是否合理（总倍数>3x需警示）
- 反稀释+对赌叠加的极端场景模拟

### 第三步：九民纪要合规检查

| 条款类型 | 合规要点 | 风险提示 |
|---------|---------|---------|
| 与目标公司对赌 | 现金补偿须在可分配利润范围内；股权回购须经合法减资 | 无可分配利润则无法执行 |
| 与股东对赌 | 原则有效，创始人/大股东回购义务可执行 | 需关注回购主体实际支付能力 |
| 公司担保 | 须经股东会决议；关联担保关联方回避 | 未经决议可能被认定无效 |
| 让与担保 | 股权让与担保效力认定趋严 | 建议改用股权质押+回购 |
| 对赌+回购竞合 | 同时触发时执行顺序和金额上限 | 建议明确约定竞合处理规则 |

### 第四步：谈判建议

根据审查视角给出差异化建议，按优先级排序：

**必争**（对结果影响大，不应让步）
**可争**（有利于己方，值得争取）
**可让**（影响有限，可作为谈判筹码交换）

**常见条款陷阱提醒**：
- 参与型清算权无cap：退出时双重获益
- 完全棘轮+对赌叠加：创始人风险极高
- 领售权无最低价格：可被低价强制退出
- 回购主体仅为目标公司：九民纪要下可能无法执行
- 竞业无补偿：可能被认定无效（劳动合同法第23条）
- 信息权范围过宽+违约金过高：可能成为GP退出工具

## 质量自检

- [ ] 条款提取覆盖四大类别（经济性/控制性/退出/其他）
- [ ] 九民纪要相关条款区分"与股东对赌"和"与目标公司对赌"
- [ ] 回购条款检查可执行性（减资程序/资金来源/支付能力）
- [ ] 反稀释区分完全棘轮和加权平均
- [ ] 投资额/估值/持股%交叉验证
- [ ] 高风险条款均有详细分析和谈判建议
- [ ] 根据审查视角给出差异化建议

## 关联技能

- **测收益** -- 用 `/测收益` 测算不同条款对回报的影响
- **投决备忘录** -- 条款审查完成后汇总进IC材料
