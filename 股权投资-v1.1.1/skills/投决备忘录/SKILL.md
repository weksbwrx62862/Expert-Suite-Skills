# /投决备忘录 -- 投委会材料生成

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

**独立能力（无需连接器）**

- 按机构标准框架生成IC Memo
- 投资逻辑四维论述 + 估值分析 + 条款摘要 + 风险矩阵 + 投资建议

**增强能力（连接器加持）**

| Connector | 增强能力 |
|-----------|---------|
| **Notion** | 投决备忘录写入 Notion 项目知识库，IC 成员协作评审 |

> 没有连接器也完全可以使用。

**重要提示**：本技能辅助材料起草，不构成投资建议。须投资总监/合伙人结合实际确认。

## 调用方式

```
/投决备忘录 <项目信息和尽调发现>
```

## 输入要求

确认：
1. **项目名称和行业**
2. **交易条款**：金额/估值/持股/投资工具
3. **尽调发现**：财务/法律/业务关键发现
4. **基金信息**：基金名称、策略、剩余可投规模

**完整度处理**：
- 完整（尽调报告+条款）--> 完整Memo
- 部分（有BP无尽调）--> "初步Memo（待补充）"
- 最小（仅名称行业）--> 仅框架+大量"[待补充]"

## 执行流程

### 第一步：项目概览

汇总：公司名称、成立时间、注册地/架构、行业、阶段、核心团队、融资历史、本轮条款。

### 第二步：投资逻辑四维论述

每维度须有数据支撑，不允许纯定性描述：

**赛道论述**：
- TAM/SAM/SOM量化、CAGR、政策方向、产业链位置
- 至少引用1个行业数据源

**公司论述**：
- 核心壁垒量化、与前2名竞对的差异化
- 至少1个可量化竞争优势指标

**时点论述**：
- 行业拐点信号、公司里程碑节点、近期催化剂
- 错过窗口的成本分析

**团队论述**：
- 核心能力匹配度、执行力证据（里程碑达成率）
- 团队稳定性（核心成员在位时间）

### 第三步：尽调关键发现

分类汇总：
- **正面发现**：支持投资决策的证据
- **风险发现**：每项标注严重度
  - **致命**（可能放弃）/ **重大**（交易前解决）/ **一般**（投后跟踪）
- **交易前提条件**：投资前必须完成事项

### 第四步：估值分析

**估值方法选择**：

| 阶段 | 主要方法 | 辅助方法 | 不适用 |
|------|---------|---------|--------|
| 天使/A轮 | 可比交易 | — | DCF |
| B轮 | 可比交易+可比公司 | 收入倍数 | DCF（无利润时） |
| C轮/Pre-IPO | 可比公司+DCF | 可比交易 | — |
| 控股收购 | DCF+可比公司 | 重置成本法 | — |

**可比公司分析**：选3-5家可比，标注选择理由，计算中位数/均值。

**估值合理性判断**：
- 本轮估值倍数 vs 可比中位数的溢价/折价比率
- 溢价合理性解释（增速溢价/稀缺性溢价/控制权溢价）

**DCF关键假设**（如适用）：
- 预测期：5年显性期 + 永续增长（2-3%）
- WACC：同行业参考12-18%（未上市流动性折价2-3%）
- 退出倍数法 vs 永续增长法交叉验证

### 第五步：交易条款摘要

列出关键条款（引用 `/审条款` 结果如有）：
- 投资金额/估值/持股/投资工具/优先权
- 对赌条款（注意九民纪要影响）
- 回购条款、董事席位、信息权
- 共售权/领售权/锁定期

### 第六步：退出预期

- 退出路径：IPO(A/H/US)/并购/S基金/回购
- 退出时间：预计X年
- 预期回报：基于退出假设的IRR/MOIC（可用 `/测收益` 详细测算）

### 第七步：投资建议

| 判断 | 条件 |
|------|------|
| 建议投资 | 四维逻辑成立，无致命风险，估值合理 |
| 附条件投资 | 逻辑成立但有重大风险需解决/估值需谈 |
| 建议放弃 | 致命风险/核心逻辑不成立 |
| 建议搁置 | 信息不足，需补充尽调后再议 |

### 第八步：输出交付

**如已连接 Notion：**
- 投决备忘录写入 Notion 项目知识库，通知链接

**如未连接：**
- Markdown输出，建议复制到Word分发IC成员

```

**日期**：[YYYY-MM-DD]  **撰写人**：[待填]

## 交易概要
| 条款 | 内容 |
|------|------|
| 投资金额 | [X]万元 |
| 估值 | Pre [X] / Post [X] |
| 持股比例 | [X%] |
| 投资工具 | [优先股/普通股] |

## 投资逻辑
[赛道+公司+时点+团队，每维度有数据]

## 尽调关键发现

## 估值分析
[方法选择+可比分析+合理性判断]

| 风险 | 严重度 | 缓释措施 |

```

## 质量自检

- [ ] 四维论述每维度有数据支撑
- [ ] 估值有可比对象（>=3个）且方法与阶段匹配
- [ ] 对赌条款考虑九民纪要影响
- [ ] 风险标注严重度（致命/重大/一般）
- [ ] 退出预期现实可行
- [ ] 投资建议有明确判断和理由

## 关联技能

- **筛项目** -- 投决前先快筛
- **尽调清单** -- 确保尽调覆盖完整
- **审条款** -- 条款审查结果纳入投决
- **测收益** -- 回报测算纳入退出分析
