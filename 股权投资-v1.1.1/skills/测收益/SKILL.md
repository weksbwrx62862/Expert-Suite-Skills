# /测收益 -- PE/VC投资收益测算

> 如遇到不熟悉的占位符或需检查已连接工具，参见 [CONNECTORS.md](../../CONNECTORS.md)。

**独立能力（无需连接器）**

- IRR/MOIC/DPI计算（含优先股瀑布分配）
- 三情景分析 + 25格敏感性矩阵（退出倍数 x 退出时间）
- 基金层面净回报（扣管理费+Carry后的LP净IRR/净MOIC）
- GP/LP瀑布分配计算

> 本技能无需连接器即可完整使用，所有功能基于用户提供的交易条款和退出假设。

**重要提示**：测算结果完全取决于用户假设参数。不构成投资建议。

## 调用方式

```
/测收益 <交易条款和退出假设>
```

## 执行流程

### 第一步：参数确认与校验

- 持股% = 投资额 / post-money估值（交叉验证）
- 退出倍数是否在行业合理范围（SaaS: PS 5-15x；消费: PE 15-30x；制造: PE 10-20x）
- 不一致时向用户确认

### 第二步：基准情景计算

**MOIC计算** = 退出分配总额 / 投资金额

退出分配按投资工具：

**普通股**：
- 退出分配 = 退出总价值 x 稀释后持股%
- 稀释后持股 = 原始持股 x (1 - 稀释比例)

**优先股（非参与型）**：
- 退出分配 = max(清算金额, 转股后分配)
- 清算金额 = 投资额 x 清算倍数
- 转股分配 = 退出总价值 x 稀释后持股%

**优先股（参与型）**：
- 退出分配 = 清算金额 + (剩余价值 x 转股比例)
- 如有cap：退出分配 = min(上述, cap金额)

**可转债**（如适用）：
- 转股前：按债权计算利息
- 转股后：按转股价计算持股，再按股权逻辑

**IRR计算**：
- 现金流：年0为-投资额，退出年为+退出分配
- IRR使净现值=0的折现率：Sum[CFt/(1+IRR)^t] = 0
- 如有中间分红/部分退出，按实际时点计入

**DPI** = 累计已分配 / 实缴资本（仅有实际分配时计算）

### 第三步：多情景分析

| 情景 | 退出倍数调整 | 退出时间 | 稀释调整 |
|------|------------|---------|---------|
| 乐观 | +20% | -1年 | -5% |
| 基准 | 用户输入 | 用户输入 | 用户输入 |
| 悲观 | -30% | +2年 | +10% |

### 第四步：25格敏感性分析（必须执行）

退出估值倍数(5梯度) x 退出时间(3-7年)，基准场景加粗。

### 第五步：GP/LP瀑布分配（如适用）

标准四层瀑布（美国模式/欧洲模式按用户选择）：

**美国模式（逐笔分配）**：
1. 返还该项目投资成本给LP
2. 优先回报（Preferred Return，通常8%）给LP
3. GP追赶（Catch-up，通常100%给GP直到GP获得总利润的20%）
4. 80/20分成（超额利润LP80%+GP20%）

**欧洲模式（整体基金）**：
1. 返还全部已投资本给LP
2. 优先回报（8%年化）给LP
3. GP追赶
4. 80/20分成

计算：
- Gross IRR/MOIC（项目层面，扣费前）
- Net IRR/MOIC（LP层面，扣管理费+Carry后）
- 管理费 = 承诺资本 x 2%/年 x 基金期限（投资期后通常按净资产计）
- Carry = (总分配 - 实缴资本 - 优先回报) x 20%

### 第六步：回购保底分析（如适用）

- 回购价 = 投资成本 + 年化X%利息（单利/复利需确认）
- 回购IRR和MOIC
- 与正常退出对比
- 九民纪要下可执行性提示

### 第七步：跨境税费（如适用）

| 架构 | 税费项 | 参考税率 |
|------|--------|---------|
| VIE/红筹退出 | 预提所得税 | 10%（税收协定可能减免） |
| 人民币基金投境外 | ODI审批+外汇限制 | — |
| 港股退出 | 资本利得 | 0%（港股暂免） |

标注"[税费估算仅供参考，建议咨询税务顾问]"

## 常见计算错误提醒

1. **稀释计算**：后续融资稀释是对所有老股东
2. **IRR与MOIC不匹配**：5年2x≈15% IRR，明显偏差需检查
3. **非参与型清算权**：取max而非加总
4. **参与型cap**：超过cap后按非参与型处理
5. **管理费基数**：投资期按承诺资本，退出期按净资产
